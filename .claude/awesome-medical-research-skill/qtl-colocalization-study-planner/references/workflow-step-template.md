# Workflow Step Template

Follow this sequence:
1. define the true trait / locus / molecular-layer question
2. choose the best-fit colocalization pattern
3. define candidate GWAS and QTL resource types
4. define tissue / cell-type / ancestry / LD requirements
5. define locus window and harmonization rules
6. select the primary colocalization method stack
7. define candidate prioritization rules
8. add follow-up modules only when justified
9. define evidence tiers and claim boundaries
10. review ambiguity, failure points, and fallback logic
11. recommend the lead protocol version

Do not skip harmonization, locus-definition, or claim-boundary review.
