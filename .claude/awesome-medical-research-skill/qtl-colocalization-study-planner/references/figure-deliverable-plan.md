# Figure and Deliverable Plan

Map the deliverables to the workload tier.

## Lite
- Study design overview
- Locus-selection and data-architecture table
- One or a few locus plots with primary coloc output
- Candidate ranking summary table

## Standard
- Lite outputs
- Multi-locus prioritization summary figure
- Tissue / QTL-layer comparison figure if relevant
- Evidence-tier table with justification notes

## Advanced
- Standard outputs
- Fine-mapping / conditional-analysis support figure where justified
- Ambiguous-locus review panel
- Linked follow-up module summary (e.g., SMR, annotation, sc-eQTL extension) if used

## Publication+
- Advanced outputs
- Multi-layer convergence figure
- Cell-type/context extension panel if used
- Translational prioritization summary with strict claim boundaries

## General Rule
Every figure should answer a real study question. Do not add ornamental figures that do not strengthen locus interpretation or candidate prioritization.
