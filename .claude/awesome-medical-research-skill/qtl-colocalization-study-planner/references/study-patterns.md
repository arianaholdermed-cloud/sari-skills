# Study Patterns

## Pattern 1 — Targeted Single-Locus Colocalization
Best when the user already has one or a few high-priority GWAS loci and wants deep interpretation.

## Pattern 2 — Candidate-Gene Prioritization Across Known GWAS Loci
Best when the goal is to rank plausible effector genes across multiple established loci without claiming genome-wide discovery.

## Pattern 3 — Trait-Centered QTL Layer Scan With Locus Follow-Up
Best when the user starts from one trait and wants to screen relevant loci using one dominant QTL layer, then prioritize only the strongest loci.

## Pattern 4 — Multi-QTL Triangulation at Shared Loci
Best when the question requires comparing eQTL, pQTL, and/or sQTL evidence at the same loci rather than relying on one molecular layer.

## Pattern 5 — Cell-Type / Context-Specific Colocalization Extension
Best when tissue-average QTL is likely insufficient and the user needs cell-type or context-specific regulatory interpretation.

## Selection Rules
- Prefer the simplest pattern that answers the user's question.
- Use targeted single-locus design when locus complexity matters more than breadth.
- Use multi-QTL triangulation only when the layers answer the same locus-level question and can be meaningfully aligned.
- Use cell-type/context extension only when ordinary bulk-tissue QTL is likely to miss the relevant biology.
- Do not default to genome-wide screening if the user's true goal is locus interpretation.
