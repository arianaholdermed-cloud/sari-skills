# Adjustment Selection Rules

## Core Rule
Adjustment must follow variable role logic, not convenience.

## Handling Categories
Use one of the following:
- must adjust / control
- recommended if available
- optional precision variable
- do not adjust
- role uncertain — requires clarification

## Important Rule
A variable may be excluded because it is a mediator, a collider, post-baseline, or otherwise likely to introduce bias when conditioned on.

## Reporting Rule
Every recommended adjustment variable should have a short rationale. Every excluded variable should also have a reason.
