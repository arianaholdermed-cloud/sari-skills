# Common Aim Failure Modes

Use this module to audit weak aim packages.

## Frequent Problems

### Vague Aim
The aim sounds important but does not define a concrete answerable question.

### Circular Hypothesis
The hypothesis simply restates the aim without adding a testable expectation.

### Outcome Drift
The stated aim and the likely measured outcome are not fully aligned.

### Overlapping Aims
Two or more aims partly answer the same question but are presented separately.

### Dependency-Heavy Aim
The aim only works if several unstated conditions succeed first.

### Confirmatory Overclaim
The aim is written as if it is confirmatory even though the logic is exploratory.

## Important Rule

If several failure modes appear, simplify before polishing. Better wording does not fix a weak aim structure.
