# Objective Operationalization Framework

Use this module to convert a broad objective into an operational one.

## Core Elements
Check whether the objective includes or clearly implies:
- target population or biological system
- exposure / biomarker / intervention / mechanism candidate
- outcome or readout
- comparator or reference context if needed
- time horizon if relevant
- setting or study context if relevant
- measurable unit of assessment

## Important Rule
Not every objective needs every element explicitly, but every refined objective must contain enough structure to guide design decisions.

## Failure Modes
- keeping the population undefined
- using broad verbs like “explore” without clarifying what will actually be evaluated
- naming a topic without defining a measurable relationship or assessment target
- mixing mechanism, validation, and outcome aims into one sentence
