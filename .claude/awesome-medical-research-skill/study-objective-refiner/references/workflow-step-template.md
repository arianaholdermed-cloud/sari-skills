# Workflow Step Template

For each workflow step in the main output, include:

1. **Step Name**
2. **Objective**
3. **What This Resolves**
4. **What Is Explicit vs Inferred**
5. **What Still Needs Narrowing**
6. **Expected Output**

## Typical Workflow Order
- interpret the real study intention
- classify objective type
- identify ambiguity and missing operational elements
- determine confirmatory versus exploratory posture
- choose the best operational structure
- rewrite the objective
- define scope boundaries
- assess measurability and executability
- recommend downstream next step
