# Population Priority Rules

Use this module to rank which subgroup gap is strongest for next-step research.

## Priority Dimensions
- biological plausibility
- clinical relevance
- evidence thinness
- likely value of resolving the gap
- feasibility of a focused follow-up study
- advantage over repeating broad-cohort work

## Rule
When several subgroup gaps are possible, rank them. Do not present them as equivalent unless the evidence truly supports equivalence.
