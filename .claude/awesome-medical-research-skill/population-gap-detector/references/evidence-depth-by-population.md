# Evidence Depth by Population

Use this module to judge how deeply the evidence base actually covers a subgroup.

## Evidence Depth Levels
1. Not studied
2. Mentioned but not analyzed
3. Descriptively reported only
4. Analyzed but underpowered or weakly interpreted
5. Signal reported without replication
6. Supported across studies but still incompletely resolved
7. Subgroup-specific effect is reasonably interpretable and externally supported

## Reporting Rule
Always separate subgroup mention, subgroup analysis, and subgroup-specific validation.
