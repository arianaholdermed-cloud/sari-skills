# Population Axis Framework

Use this module to define which population axes are relevant before judging subgroup gaps.

## Core Population Axes
- age
- sex
- geography
- ancestry
- comorbidity
- disease stage
- treatment line
- exposure history
- molecular subtype
- tissue context
- care setting
- special populations

## Rule
Only include axes that could plausibly change biological interpretation, clinical utility, implementation, or transportability.

## Important Rule
Do not list every possible subgroup axis by default. Relevance matters more than exhaustiveness.
