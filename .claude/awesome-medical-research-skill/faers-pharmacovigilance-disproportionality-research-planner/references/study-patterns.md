# Study Patterns — Detailed Logic
# faers-pharmacovigilance-disproportionality-research-planner

---

## Pattern A — Single-Drug Disproportionality Workflow

**Use when:** User wants one product benchmarked against one comparator or control background.

## Pattern B — Multi-Drug Class Comparison Workflow

**Use when:** User wants several products within one class compared systematically.

## Pattern C — Indication-Stratified Workflow

**Use when:** User wants cases separated by disease / reason-for-use strata.

## Pattern D — Comparator-Controlled Signal Screening Workflow

**Use when:** User wants explicit therapeutic comparators or controls.

## Pattern E — Signal-Prioritization and Follow-Up Workflow

**Use when:** User wants strongest signals filtered for downstream follow-up relevance.

---

## Pattern Combination Rules

| Combination | When Appropriate |
|---|---|
| B + C | multi-drug comparison with indication stratification |
| B + D | within-class signal comparison plus explicit therapeutic controls |
| C + D | subgroup-based signal comparison with comparator benchmarking |
| D + E | comparator-controlled screen followed by strong-signal prioritization |
| B + C + E | class comparison + stratification + prioritized follow-up shortlist |
