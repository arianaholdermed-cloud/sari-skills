# Target Trial Emulation Rules

## Core Rule
Use target-trial-emulation framing only when the user’s question is treatment-strategy oriented or causal-leaning, and when explicit trial components can be mapped.

## Trial Components to Map
- eligibility criteria
- treatment strategies
- assignment moment / time zero
- follow-up start and end
- outcomes
- causal contrast of interest
- analysis plan aligned to assignment and follow-up logic

## Important Rule
If key trial components cannot be approximated because of unmeasured severity, delayed treatment capture, poor adherence measurement, or unresolved switching logic, say so clearly.

## Reporting Rule
Classify the output as:
- target trial emulation recommended
- target trial emulation partially approximable
- target trial emulation not appropriate
