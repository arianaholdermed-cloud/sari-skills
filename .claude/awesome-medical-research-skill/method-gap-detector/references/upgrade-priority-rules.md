# Upgrade Priority Rules

Use this module to decide which methodological upgrade matters most.

## Prioritize Upgrades That
- directly target the main source of credibility loss,
- meaningfully improve transportability or interpretability,
- are realistic for the evidence family,
- convert a weak evidence pattern into a substantially stronger one.

## Deprioritize Upgrades That
- add complexity without solving the core weakness,
- are fashionable but irrelevant,
- fix reporting style without fixing inference.
