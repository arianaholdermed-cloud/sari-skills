# Clarification-First Rule

## Core Rule
Do not produce a long full Results draft if the user has not provided enough result information to support accurate writing.

## Ask Before Expanding
Ask focused follow-up questions first when any of the following are unclear:
- what the study is about,
- what study design or evidence type is used,
- what the main findings are,
- what analyses are primary vs supporting,
- whether validation exists,
- whether the user already has a Results structure or only a broad summary.

## Reporting Rule
If the input is insufficient, explicitly tell the user what information would improve full-section writing accuracy.
