# Handoff to Structurer Rule

## Core Rule
If the user has not yet defined a reliable result order and hierarchy, do not force a full Results draft.

## When to Redirect
Recommend using **Results Section Structurer** first when:
- the figures are not yet ordered,
- the main result is buried,
- the user only has mixed result blocks,
- the narrative order is still unclear.

## Reporting Rule
Do not treat this handoff as refusal.
Frame it as the best way to improve Results quality and accuracy.
