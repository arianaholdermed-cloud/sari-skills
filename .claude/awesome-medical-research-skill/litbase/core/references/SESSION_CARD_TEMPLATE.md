## How to use this card

1. **First setup**: Run `/setup` in a Web Claude conversation. At the end, you will receive a filled-in version of this card based on your answers.

2. **Each new session**: Paste the card at the start of the conversation before running any command. This gives me the context I need.

3. **After each session**: I will output an updated card at the end of the conversation. Copy it and replace your saved card with the new version.

4. **Commands work the same way**: Type `/feed`, `/read`, `/discuss`, `/recap`, `/update`, or `/propose` in the conversation. Describe what you want to do and I will execute the corresponding workflow.

5. **Reading a paper**: Upload the PDF or paste the abstract + basic info (title, authors, journal, year, DOI if available). Then type `/read`.
