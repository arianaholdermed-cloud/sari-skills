# LitBase Workflow

Every `.md` note in this folder is a structured analysis report generated by **Claude Code** after reading the original PDF.

---

## Available Commands

Open Claude Code in this folder and use:

| Command | Function |
|---------|----------|
| `/read` | Deep-read a paper; generate a structured 4-section note |
| `/discuss keyword` | Discuss an already-read paper; auto-records key insights to the note |
| `/feed` | Recommend new papers; list open-access PDFs ready to read |
| `/recap` | View reading progress and theoretical framework completeness |
| `/update` | Sync updates when your research direction changes |
| `/sync` | Check all documents for consistency; auto-fix format issues |
| `/setup` | First-time setup: configure research profile and search terms |
| `/propose` | Build a Research Foundation Document from accumulated reading notes |

---

## How to Submit a Paper

Type `/read`, then submit using any of the following methods:

### Method 1: Local PDF (recommended)
Provide the path:
```
/Users/yourname/Downloads/paper.pdf
```

### Method 2: DOI or journal URL
```
https://doi.org/10.xxxx/xxxxx
```
Open-access papers can be fetched directly; paywalled papers require a local PDF.

### Method 3: Abstract + basic metadata
Paste the abstract and provide the journal name and year.

---

## What Claude Outputs

Each paper analysis contains four sections:

| Section | Content |
|---------|---------|
| **I. Paper Weight** | Journal ranking (SSCI/AHCI/IF), citation count, author academic background (h-index, institution, research focus) |
| **II. Paper Highlights** | Methodological innovation / critique of systemic problems / significance of the research object |
| **III. Transferable Elements** | Theoretical frameworks, method details, conceptual tools |
| **IV. How to Use in Your Paper** | Literature review positioning, suggested citation phrases, methodological precedent, research motivation |

Analysis language: **English primary.**

---

## Paper Recommendation System

Type `/feed` — Claude updates search terms and recommends papers across three tiers. Open-access papers are noted with PDF URLs; paywalled papers include DOI links and access paths.

---

## PDF Auto-Rename

After downloading a new PDF, place it in `data_dir`, then run:

```bash
python scripts/rename_pdfs.py --dry-run   # preview, no actual renaming
python scripts/rename_pdfs.py             # execute renaming
```

---

## Metadata Query Tool

```bash
python scripts/lookup_paper.py --doi "10.xxxx/xxxxx"
python scripts/lookup_paper.py --title "paper title keywords"
```

## Reading Progress Snapshot

Run `/recap` to generate a framework completeness analysis, auto-saved to `notes/recaps/YYYY-MM-DD_recap.md`.
