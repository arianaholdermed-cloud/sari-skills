# Constraint-to-Design Adjustment Rules

Use this module to translate real constraints into design modifications.

## Common Adjustment Moves
- narrow the endpoint
- narrow the population
- reduce assay complexity
- reduce validation burden
- replace unavailable wet-lab work with computational analysis
- defer prospective collection
- defer external validation
- simplify the model burden
- convert a mixed design into a single lead route

## Important Rule
Every major design adjustment must be tied to a concrete feasibility constraint.

## Reporting Rule
State both the adjustment and the constraint that forced it.
