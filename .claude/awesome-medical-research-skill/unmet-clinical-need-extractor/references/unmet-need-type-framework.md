# Unmet Need Type Framework

Use this module to classify the extracted unmet clinical needs.

Classification improves precision and prevents different care problems from being merged into vague language.

## Common Unmet-Need Types

### Screening / Early-Detection Gap
Current tools detect disease too late, too inconsistently, or too impractically.

### Diagnostic Gap
Current diagnostic workflows are delayed, inaccurate, invasive, ambiguous, or poorly scalable.

### Stratification Gap
Current tools do not adequately define meaningful subgroups, severity groups, or management-relevant risk classes.

### Treatment-Selection Gap
Current decision-making does not reliably identify who should receive which treatment.

### Response-Prediction Gap
Current tools do not predict benefit, resistance, or durability well enough for real decision use.

### Monitoring Gap
Current tools do not adequately detect treatment response, residual disease, recurrence, or progression.

### Toxicity / Trade-Off Gap
Care decisions are limited by poor ability to balance efficacy, harm, de-escalation, or overtreatment risk.

### Implementation / Access Gap
Useful tools or pathways exist but are not sufficiently deployable, accessible, standardized, or adopted.

## Important Rules

- Do not merge biomarker desire with treatment-selection need unless the actual clinical decision gap is clear.
- Distinguish response prediction from prognosis.
- Distinguish monitoring need from baseline diagnosis need.
- Use the most decision-relevant category available.
