# Reproducibility Gap Rules

Flag what blocks faithful reproduction.

## Common gap classes
- missing dataset accession or source details
- missing inclusion/exclusion logic
- unclear preprocessing
- absent software/package versions
- absent parameter settings
- unavailable code
- unavailable raw data or materials
- unclear validation workflow
- unreported analyst decisions

## Rules
1. Treat each missing operational dependency as a concrete reproducibility gap.
2. Rank gaps by likely impact on successful reproduction.
3. Distinguish manageable assumptions from replication-stopping gaps.
