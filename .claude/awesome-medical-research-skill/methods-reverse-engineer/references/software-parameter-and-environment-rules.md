# Software, Parameter, and Environment Rules

Capture reproducibility-critical technical details only when supported.

## Extract when reported
- software names
- package names
- version numbers
- operating environments if relevant
- assay platforms / instruments / kits
- thresholds / cutoffs / hyperparameters
- statistical significance thresholds and correction rules

## Rules
1. Never invent versions.
2. Separate reported parameters from likely but unreported parameters.
3. Highlight missing tool or environment details when they affect reproducibility.
