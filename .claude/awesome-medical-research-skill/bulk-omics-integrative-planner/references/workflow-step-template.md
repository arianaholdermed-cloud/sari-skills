# Workflow Step Template

Use a dependency-consistent workflow.

## Workflow Shape
1. define question, comparison structure, and dominant omics layer(s)
2. identify candidate datasets / cohorts / sample sources
3. define metadata and covariate requirements
4. perform preprocessing, QC, and normalization
5. run the primary contrast or stratification analysis
6. perform pathway / functional interpretation
7. add clinical association or integrative analysis only if justified by the question and metadata
8. perform external support / validation steps
9. define translational or experimental extension if appropriate

## Mandatory Disclaimer Rule
If step 2 or any later step names datasets, cohorts, repositories, databases, or public resources, place the mandatory Dataset Disclaimer immediately before step 1.
