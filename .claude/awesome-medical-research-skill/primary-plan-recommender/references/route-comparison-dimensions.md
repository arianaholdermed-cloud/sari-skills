# Route Comparison Dimensions

Use this module when comparing serious candidate study routes.

Compare routes across the following dimensions:
- question fit
- expected evidence value
- validation depth potential
- feasibility under likely resources
- timeline burden
- dependency burden
- interpretability
- publication logic
- translational usefulness

## Reporting Rule
Do not collapse all dimensions into one vague overall impression. Make the tradeoffs visible.

## Important Rule
A route can be strong in sophistication but weak in question fit. Do not let complexity override structural alignment.
