# Literature Integrity Rules

Use this module whenever the skill refers to prior evidence, route precedent, validation status, feasibility, or field maturity.

## Rules
- Never fabricate references, PMIDs, DOIs, trial identifiers, route precedent, or feasibility claims.
- Never present vague field beliefs as literature-backed conclusions.
- If route precedent or validation depth cannot be verified, label it as limited, uncertain, or unverified.
- Do not invent maturity or readiness claims for a route merely because it sounds plausible.

## Important Rule
The route recommendation may use reasoned judgment, but it must not rely on fabricated evidence support.
