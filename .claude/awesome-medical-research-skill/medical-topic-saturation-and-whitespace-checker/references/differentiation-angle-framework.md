# Differentiation Angle Framework

Use this module to construct viable entry angles when a topic is crowded but not fully closed.

A differentiated angle should improve claim quality, relevance, or translation potential.
It should not merely change surface presentation.

## Common Differentiation Axes

- better-matched population or disease stage
- clinically stronger endpoint
- higher-quality comparator design
- stronger external validation path
- cleaner assay / platform alignment
- translationally more useful framing
- implementation-aware workflow angle
- mechanism-supported rather than purely associative design
- orthogonal evidence bridge

## Strong Differentiation Profile

A strong angle usually has all of the following:
- materially different claim value,
- believable execution path,
- reduced direct overlap with the most crowded subspace,
- and a clearer reason for publication or adoption interest.

## Weak Differentiation Profile

A weak angle usually has one or more of the following:
- only superficial rebranding,
- same core template as prior studies,
- no stronger validation or comparator plan,
- no clinically meaningful repositioning,
- no real reduction in crowding.

## Important Rule

When recommending an entry angle, name the angle directly and state why it remains differentiable under current field conditions.
