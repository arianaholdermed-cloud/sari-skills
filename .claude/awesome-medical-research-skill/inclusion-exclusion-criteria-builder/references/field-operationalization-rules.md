# Field Operationalization Rules

## Core Rule
Major eligibility criteria should be tied to a plausible evidence source.

## Possible evidence sources
- diagnosis code or clinician-documented diagnosis
- pathology report
- imaging report
- medication administration or prescription record
- procedure record
- laboratory value with timestamp
- operative note
- registry variable
- specimen inventory / biobank field

## Reporting rule
If the exact field is unknown, use a generic but plausible label such as “pathology-confirmed diagnosis” or “documented systemic therapy start,” and explicitly avoid pretending that a specific database field is known.
