# Population Framing Rules

## Core Rule
Eligibility criteria should start from an operational source population, not an abstract disease label.

## Required elements
Define as many of the following as the user has made available:
- condition / disease entity
- setting (hospital, ICU, outpatient clinic, registry, pathology archive, biobank)
- study design context
- age frame if relevant
- disease stage, severity, subtype, or therapy line if relevant
- specimen or test availability if relevant
- key index moment or screening moment

## Reporting rule
When assumptions are required, surface them in Section A rather than embedding them silently into criteria.
