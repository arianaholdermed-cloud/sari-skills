# Clarification-First Rule

## Core Rule
Do not produce a long LaTeX conversion plan if the source manuscript or target template information is still too incomplete.

## Ask Before Expanding
Ask focused follow-up questions first when any of the following are unclear:
- what the source format is,
- what the target template or venue is,
- whether the scope is full-manuscript or partial conversion,
- whether bibliography and figure assets exist,
- whether the user wants planning, cleanup, or direct conversion output.

## Reporting Rule
If the input is insufficient, explicitly tell the user what information would improve conversion accuracy.
