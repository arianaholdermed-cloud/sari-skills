# Compile Boundary Rules

## Core Rule
A structurally converted LaTeX manuscript is not automatically a compile-ready submission package.

## Common Missing Assets
- `.cls` files
- `.bst` files
- bibliography databases
- figure assets
- supplement files
- journal macros

## Important Rule
Do not overpromise compile success when essential assets are missing.

## Reporting Rule
Separate structural conversion readiness from compile readiness.
