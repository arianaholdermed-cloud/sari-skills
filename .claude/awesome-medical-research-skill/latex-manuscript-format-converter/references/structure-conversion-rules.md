# Structure Conversion Rules

## Core Rule
Manuscript components should be mapped into LaTeX structure systematically.

## Common Components
- title page
- author and affiliation block
- abstract
- keywords
- section hierarchy
- figures and tables
- references
- supplementary materials
- equations

## Important Rule
The main task is structural normalization, not scientific rewriting.

## Reporting Rule
Make the source-to-LaTeX component mapping explicit.
