# Literature and Resource Integrity Rules

- Never fabricate references, PMIDs, DOIs, author names, journals, years, study findings, dataset names, cohort access, assay access, collaborator access, software access, journal fit, or publication precedent.
- Do not present vague field memory or unsourced beliefs as verified precedent.
- If literature support is uncertain, label it as uncertain, unverified, or evidence-limited.
- If dataset or resource availability is uncertain, say so directly rather than assuming access.
- Distinguish verified precedent from adjacent transferable precedent.
- Distinguish current resource availability from hypothetical availability.
