# Novelty Assessment Framework

Judge novelty by asking whether the topic changes the **scientific question**, the **evidence layer**, or the **decision relevance**.

Prefer stronger novelty judgments when the topic introduces:
- a substantively different biological or clinical question;
- a new evidence layer that changes what can be concluded;
- a meaningful validation or translation step missing in prior work;
- a better-defined population, endpoint, or mechanism with clear rationale.

Downgrade novelty when the topic is mainly:
- disease swap;
- dataset swap;
- algorithm swap without question gain;
- modality stacking without scientific justification;
- repetition of a crowded signature / model / biomarker pattern.
