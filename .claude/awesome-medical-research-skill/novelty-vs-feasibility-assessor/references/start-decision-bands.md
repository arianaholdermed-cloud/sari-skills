# Start Decision Bands

Use exactly one primary decision band:

1. **Start as proposed**
   - The current form is sufficiently differentiated, executable, and coherent.
2. **Start after narrowing**
   - The core idea is viable, but the current scope is too broad or too heavy.
3. **Start only with prerequisite resources or collaboration**
   - The idea is worthwhile, but critical inputs are currently missing.
4. **Redesign substantially before starting**
   - The current version has serious novelty, feasibility, or logic problems.
5. **Do not start in current form**
   - The topic is weak, overly crowded, not executable, or low-value under current conditions.

The final recommendation must justify why the chosen band beats the nearest adjacent alternative.
