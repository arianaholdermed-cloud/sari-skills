# Scope Framing Rules

Use this module to fix the scan boundary before evidence mapping.

Always define:
- disease / indication / subtype
- focal entity level: drug, target, target class, pathway, mechanism class
- whether the question is biology-first, development-first, or competition-first
- modality scope
- whether the user wants broad mapping or specific target diligence

Do not let the scan remain so broad that target-level judgments become vague.
