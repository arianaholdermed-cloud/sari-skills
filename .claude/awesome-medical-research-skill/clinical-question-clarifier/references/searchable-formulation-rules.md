# Searchable Formulation Rules

A searchable formulation should be concise, concept-based, and retrieval-ready.

## Include
- disease / condition
- population or context if essential
- key intervention / exposure / biomarker / mechanism
- main outcome or objective
- special boundary if central (stage, subtype, treatment setting)

## Avoid
- overlong conversational phrasing
- multiple unrelated aims in one query
- hidden assumptions not reflected in the wording

## Good Searchable Form
- gastric precancerous lesions early intervention targets network pharmacology
- sepsis biomarker baseline predictor mortality adults ICU
- lupus single-cell pathway activity cell-state heterogeneity

## Rule
The searchable version should be specific enough to retrieve relevant literature but broad enough to avoid premature overrestriction.
