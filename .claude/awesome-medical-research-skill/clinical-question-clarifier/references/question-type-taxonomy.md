# Question Type Taxonomy

Classify the dominant question before selecting a framing structure.

## 1. Treatment / Intervention
Use when the user is asking whether an intervention helps, harms, or performs better than another option.

## 2. Diagnosis / Diagnostic Test Accuracy
Use when the focus is on identifying a condition, test performance, or diagnostic comparison.

## 3. Prognosis
Use when the question asks whether a baseline factor predicts a future outcome.

## 4. Prediction / Biomarker Stratification
Use when the user wants to classify, stratify, or predict response, risk, or subgroup assignment.

## 5. Exposure / Risk / Epidemiology
Use when the user is asking about association between an exposure and an outcome.

## 6. Causality / Etiology
Use when the user wants causal interpretation rather than simple association.

## 7. Mechanism / Biology
Use when the real question is about pathway, cell type, biological process, or mechanistic explanation.

## 8. Implementation / Health Services
Use when the issue is adoption, workflow, real-world use, barriers, or service delivery.

## 9. Translational
Use when a biological finding is being connected to a practical clinical or diagnostic use-case.

## 10. Exploratory Research-Planning
Use when the user has a broad research theme but not yet a formal question.

## Rule
If multiple types are present, identify:
- dominant question type
- secondary question type(s)
- what would change if the framing switched to the secondary type
