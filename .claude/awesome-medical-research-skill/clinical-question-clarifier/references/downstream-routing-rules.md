# Downstream Routing Rules

Question clarification should end with a recommendation for the next best workflow.

## Route to Evidence Review
Use when the clarified question is primarily about what current evidence says.

## Route to Gap Finder
Use when the question is clarified but the user needs to know what remains unresolved in the field.

## Route to Gap-to-Study Planner
Use when the user already has a defined gap or a tightly scoped study question and wants a research design.

## Route to Algorithm Matcher
Use when the user already has a study concept and needs method or algorithm selection.

## Rule
Do not leave the user with a clarified question but no next-step recommendation.
