# Framing Framework Library

Choose the framework that best matches the question type.

## PICO
Best for:
- treatment
- intervention comparison
- comparative effectiveness

Core elements:
- Population
- Intervention
- Comparator
- Outcome

## PECO
Best for:
- exposure questions
- risk factors
- epidemiologic association

Core elements:
- Population
- Exposure
- Comparator
- Outcome

## PICOTS
Best for:
- questions where timing, setting, or study type must be explicit

Core elements:
- Population
- Intervention / Exposure
- Comparator
- Outcome
- Time
- Setting / Study design

## Diagnostic Framing
Best for:
- diagnostic accuracy
- screening
- test comparison

Core elements:
- target condition
- intended population
- index test
- reference standard
- accuracy outcome

## Prognostic Framing
Best for:
- baseline factor predicting future outcome

Core elements:
- population
- prognostic factor
- outcome
- time horizon
- context / treatment status

## Mechanistic Framing
Best for:
- pathway questions
- cell-state questions
- perturbation or biological process questions

Core elements:
- biological system / disease context
- mechanism candidate
- level of observation
- comparison context
- readout or mechanistic consequence

## Implementation Framing
Best for:
- workflow adoption
- barriers / facilitators
- service delivery

Core elements:
- setting
- stakeholders
- intervention / change process
- implementation barrier or facilitator
- implementation outcome

## Translational Framing
Best for:
- converting a biological observation into a clinically meaningful question

Core elements:
- biological candidate
- intended clinical use-case
- target population
- validation need
- boundary of intended claim

## Rule
Do not select a framework just because it is familiar. Select it because it clarifies the real question with the least distortion.
