# Question Rewrite Rules

The goal is not elegant paraphrase alone. The goal is usable question structure.

## Required Versions
Always provide at least:
1. plain-language clarified version
2. research-ready version
3. searchable version

## Plain-Language Version
Should preserve the user’s intent and be easy to understand.

## Research-Ready Version
Should have enough structure to guide a study, review, or protocol.

## Searchable Version
Should be compact enough to drive literature retrieval and concept searching.

## Rewrite Rule
Do not over-specify details that the user did not imply unless doing so is necessary for usability. If you make a necessary narrowing assumption, state it explicitly.

## Failure Modes
- rewriting without narrowing
- polishing wording while leaving ambiguity intact
- changing the real question type during rewrite
