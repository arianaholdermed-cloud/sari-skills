# Figure and Deliverable Plan
# medical-research-gap-to-study-planner

---

## Core Principle

Figures should mirror the logic of gap closure, not just list analyses in chronological order.
The deliverables should show how the protocol moves from unresolved problem to evidence-generating answer.

---

## Common Figure Architecture

### Figure 1. Study Logic Overview
- gap statement
- selected study pattern
- cohorts / data / assay overview
- core evidence chain

### Figure 2. Discovery or Core Resolution Result
- the first analysis that directly addresses the central gap component

### Figure 3. Validation / Replication Layer
- independent cohort, orthogonal data, or robustness support

### Figure 4. Context / Mechanism / Translation Layer
- only if directly required by the chosen pattern

### Figure 5. Integrated Evidence and Claim Boundary Figure
- what the study supports
- what remains unresolved
- concise model or summary schematic

---

## Deliverables

A good protocol output should usually include:
- main figure plan
- supplementary analysis list
- final result table / endpoint table
- one integrated limitations summary

Do not force a long figure list if the minimal plan is intentionally compact.
