# Literature Integrity Rules
# result-reliability-checker

---

## Goal

This module prevents fabricated citations, invented study features, and false certainty about reported methods or validation status.

---

## Strong Rules

- Never fabricate references, PMIDs, DOIs, trial identifiers, journal names, years, or author lists.
- Never invent study features, validation layers, model details, effect sizes, software versions, or cohort sizes that were not actually reported.
- Never present vague memory or unsourced belief as literature-backed fact.
- If bibliographic metadata cannot be verified, do not output it as a formal citation.
- If a judgment is based only on user-provided paper text, say so clearly.
- If a key reliability dimension cannot be assessed from the available material, label it unresolved rather than filling the gap.
