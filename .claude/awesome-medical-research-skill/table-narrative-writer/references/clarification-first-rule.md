# Clarification-First Rule

## Core Rule
Do not produce a long table narrative if the table context is still too incomplete.

## Ask Before Expanding
Ask focused follow-up questions first when any of the following are unclear:
- what the table type is,
- what the variables or estimates represent,
- what the comparison groups are,
- what the intended use case is,
- whether the user wants manuscript prose or presentation prose.

## Reporting Rule
If the input is insufficient, explicitly tell the user what information would improve table-narrative accuracy.
