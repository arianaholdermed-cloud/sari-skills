# Table Message Extraction Rules

## Core Rule
A table narrative should reflect the table’s real scientific contribution, not its full visual inventory.

## Extract First
- what the table is for,
- what the main pattern is,
- what the key contrast or estimate is,
- and why the table exists in the manuscript.

## Important Rule
A table that exists mainly for transparency does not always require heavy main-text narration.

## Reporting Rule
State the table’s main message before drafting prose.
