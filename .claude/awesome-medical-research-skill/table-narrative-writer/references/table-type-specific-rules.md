# Table-Type-Specific Rules

## Core Rule
Different table types require different narrative strategies.

## Baseline Tables
Emphasize overall comparability, imbalance patterns, or notable descriptive features.
Do not overinterpret baseline differences.

## Regression Tables
Emphasize the primary estimates and their direction, magnitude, and uncertainty in a selective way.

## Subgroup Tables
Emphasize only clearly relevant subgroup patterns and avoid inflating subgroup noise.

## Model-Performance Tables
Emphasize meaningful comparison and performance hierarchy, not every metric in full.

## Sensitivity Tables
Emphasize robustness pattern rather than row-by-row restatement.

## Reporting Rule
State the table type explicitly before narrating it.
