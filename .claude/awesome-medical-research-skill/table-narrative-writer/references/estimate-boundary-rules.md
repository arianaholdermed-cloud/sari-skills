# Estimate Boundary Rules

## Core Rule
The prose must not imply more than the table supports.

## Keep Separate
- descriptive summary vs effect claim
- association vs causation
- subgroup pattern vs confirmed heterogeneity
- model metric vs clinical usefulness
- robustness support vs definitive proof

## Important Rule
The narrative should match the estimate type and table purpose.

## Reporting Rule
State when the table supports only descriptive or associative wording.
