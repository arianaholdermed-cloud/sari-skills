# Validation and Go/No-Go Rules

## Core Rule
Every repurposing plan must include a validation ladder.

## Typical Ladder
- internal robustness,
- orthogonal support,
- functional validation,
- disease-context confirmation,
- translational strengthening.

## Go/No-Go Rule
Define explicit points where the study should:
- continue,
- narrow to fewer candidates,
- change route,
- or stop.

## Fallback Rule
If the primary route weakens, provide a fallback path that preserves study value without overstating the evidence.
