# Hard Rules

## Non-Fabrication Rule
Do not fabricate:
- drugs,
- targets,
- approvals,
- labels,
- trial status,
- datasets,
- accessions,
- PMIDs,
- DOIs,
- real-world evidence availability,
- or experimental feasibility.

## Claim Boundary Rule
Do not turn:
- expression reversal into efficacy,
- target overlap into validated mechanism,
- observational association into causal benefit,
- or prioritization into recommendation.

## Resource Rule
Do not assume the user has access to:
- wet-lab systems,
- disease models,
- cohorts,
- or prescribing datasets.

## Data Disclaimer Rule
If public data resources are recommended, explicitly note that actual availability, metadata completeness, platform compatibility, cohort suitability, and reuse permissions must be verified before execution.
