# Priority Staging Rules

## Core Rule
Revision work should be staged in the order that most protects the paper.

## Typical Priority Order
1. core scientific salvage
2. missing analysis or evidence reinforcement
3. figure / structure repair
4. wording / framing / clarity polish
5. rebuttal alignment

## Important Rule
Do not spend disproportionate effort on wording before the major scientific pressure points are addressed.

## Reporting Rule
The revision plan should read like a staged rescue sequence, not a flat checklist.
