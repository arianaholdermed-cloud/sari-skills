# Clarification-First Rule

## Core Rule
Do not produce a long revision strategy if the actual review package is still too incomplete.

## Ask Before Expanding
Ask focused follow-up questions first when any of the following are unclear:
- what the decision type is,
- what the reviewer comments actually say,
- whether there is an editor letter,
- what the manuscript type and study design are,
- whether requested new work is feasible,
- whether the user wants triage, planning, or rebuttal preparation.

## Reporting Rule
If the input is insufficient, explicitly tell the user what information would improve revision-planning accuracy.
