# Comment Triage Rules

## Core Rule
Reviewer comments must be triaged by function and strategic weight, not just listed.

## Common Comment Types
- fundamental scientific concern
- missing analysis
- missing validation
- design or cohort concern
- overclaiming concern
- interpretation concern
- presentation or clarity issue
- figure or structure issue
- citation or context issue

## Triage Dimensions
Assess:
- scientific severity,
- editorial sensitivity,
- revision burden,
- impact on credibility,
- whether the issue is central or peripheral.

## Important Rule
A short comment can still be strategically critical.
A long comment is not automatically the most important one.

## Reporting Rule
The triage summary should make clear which comments truly threaten acceptance and which are repairable with lighter intervention.
