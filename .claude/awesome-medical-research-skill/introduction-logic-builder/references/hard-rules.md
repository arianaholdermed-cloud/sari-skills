# Hard Rules

1. Never invent literature, PMIDs, DOIs, guidelines, or consensus positions.
2. Never strengthen the study contribution beyond what the input supports.
3. Never create a fake “clear gap” from insufficiently defined material.
4. Never confuse literature quantity with introduction quality.
5. Never preview results in a way that distorts the study’s actual evidence level.
6. Never overstate novelty to compensate for weak positioning.
7. If the input is insufficient, explicitly say what information is needed before accurate logic building.
8. Always explain the logic-building rationale.
9. Never fabricate cohort details, validation status, or journal expectations.
