# Clarification-First Rule

## Core Rule
Do not produce a long introduction-logic output if the user has not provided enough study information to support accurate problem-gap-objective positioning.

## Ask Before Expanding
Ask focused follow-up questions first when any of the following are unclear:
- what the study is about,
- what study design or evidence type is used,
- what problem the study addresses,
- what the actual unresolved limitation or gap is,
- what this study contributes,
- whether the user is providing a draft introduction, a study summary, or only a broad topic.

## Reporting Rule
If the input is insufficient, explicitly tell the user what information would improve logic-building accuracy.
