# Logic Reporting Rule

## Core Rule
Every meaningful logic-building choice should be explained.

## Explain In Terms Of
- sharper problem framing,
- more relevant background selection,
- clearer gap definition,
- better study-position alignment,
- reduced literature-dump behavior,
- stronger narrative control.

## Important Rule
Do not say only that the logic is “clearer” or “stronger.”
State why the change matters.
