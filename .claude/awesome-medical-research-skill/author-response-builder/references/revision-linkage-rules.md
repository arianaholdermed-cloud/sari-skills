# Revision Linkage Rules

## Core Rule
A strong reviewer response should map clearly to actual manuscript changes.

## Good Linkage Targets
- revised wording
- added paragraph
- added analysis
- revised figure or table
- supplementary addition
- limitation statement
- explicit statement that no manuscript change was made and why

## Important Rule
A response without revision linkage often sounds weaker than it is.

## Reporting Rule
State what changed in the manuscript, or state transparently when no direct change was made.
