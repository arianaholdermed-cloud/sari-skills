# Unresolved Issue Rules

## Core Rule
When a request cannot be fully satisfied, the response must remain transparent.

## Good Unresolved-Issue Handling
- state what was feasible
- state what remains limited
- show how the manuscript now reflects the limitation
- avoid pretending the issue vanished

## Important Rule
A partial response can still be strong if it is honest and well bounded.

## Reporting Rule
Explicitly distinguish partial resolution from full resolution.
