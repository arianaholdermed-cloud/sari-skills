# Handoff to Logic Builder Rule

## Core Rule
If the user has not yet defined a reliable problem-gap-objective structure, do not force a full Introduction draft.

## When to Redirect
Recommend using **Introduction Logic Builder** first when:
- the topic is broad,
- the gap is still unclear,
- the study contribution is not yet well positioned,
- the user has only a direction, not a real introduction logic.

## Reporting Rule
Do not treat this handoff as refusal.
Frame it as the best way to improve writing accuracy and introduction quality.
