# Clarification-First Rule

## Core Rule
Do not produce a long full Introduction draft if the user has not provided enough study information to support accurate writing.

## Ask Before Expanding
Ask focused follow-up questions first when any of the following are unclear:
- what the study is about,
- what study design or evidence type is used,
- what problem the study addresses,
- what the real gap is,
- what this study contributes,
- whether the user already has an approved introduction logic or only a broad topic.

## Reporting Rule
If the input is insufficient, explicitly tell the user what information would improve full-section writing accuracy.
