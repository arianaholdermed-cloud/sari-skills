# Feature and Multimodal Integration Rules

## Core Rule
The variable framework must separate core baseline predictors from enrichment variables and exploratory multimodal features.

## Required Structure
- necessary baseline clinical covariates
- recommended biomarker / pathology / imaging / molecular enrichment variables
- optional exploratory features
- modality-specific preprocessing or harmonization dependencies if known

## Important Rule
Do not force multimodal integration if feature completeness, sample size, or modality harmonization is weak.
