# Treatment Context and Cohort Architecture Rules

## Core Rule
The cohort backbone must anchor the predictor to a clearly defined treatment context, therapy line, disease setting, and baseline measurement window.

## Required Behaviors
- define source population
- define treatment regimen, class, or combination context
- define line of therapy and treatment setting
- define baseline window before treatment start or decision point
- state whether the design is retrospective or prospective
- separate actually available cohorts from assumed validation cohorts

## Important Rule
Do not pool heterogeneous therapy contexts into one primary predictive design unless the rationale and control strategy are explicit.
