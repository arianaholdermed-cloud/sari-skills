# Predictive Question Fit Rules

## Core Rule
Only treat the request as a treatment-response prediction design if the intended use is to estimate likelihood of response, resistance, durable benefit, or treatment failure under a defined therapy or therapy class.

## Must Distinguish
- predictive vs prognostic
- treatment-specific response prediction vs generic outcome association
- baseline predictor vs post-treatment signal
- response endpoint vs long-term prognosis endpoint

## Important Rule
If treatment specificity is missing, the design cannot be finalized as predictive.
