# Workflow Step Template

## Standard Flow
1. clarify predictive fit
2. define treatment context and cohort backbone
3. define response / resistance endpoint
4. select study family
5. review baseline comparability and heterogeneity
6. define predictor framework
7. build model-development line
8. define validation architecture
9. audit leakage and overfitting
10. review translation readiness
11. recommend the lead protocol version

## Important Rule
Do not skip endpoint definition, baseline timing, or validation architecture.
