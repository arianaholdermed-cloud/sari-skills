# Output Section Guidance

## Core Rule
Keep the output sectioned exactly as A–L. Each section must do one job and must not collapse into a manuscript-like narrative.

## Table Priority Sections
Use tables when helpful in:
- treatment context and cohort backbone
- responder / resistance endpoint framework
- candidate predictor and variable framework
- model development plan
- validation strategy
- translation readiness and feasibility

## Important Rule
Always keep the lead protocol version explicit.
