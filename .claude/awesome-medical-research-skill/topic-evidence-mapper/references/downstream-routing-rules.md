# Downstream Routing Rules

## Purpose
Recommend the most useful next step after mapping.

## Typical Routes
- Go to literature reading when the user needs depth on a specific stream.
- Go to gap finding when the user wants high-confidence opportunity identification.
- Go to study planning when the user already knows the desired entry point.
- Go to algorithm matching when the user already has a clear question and wants method upgrading.

## Rule
The route recommendation should be justified by the map, not given generically.
