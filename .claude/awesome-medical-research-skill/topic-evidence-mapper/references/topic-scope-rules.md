# Topic Scope Rules

## Purpose
Ensure the topic is appropriately bounded before evidence mapping begins.

## Default Checks
Assess whether the topic is too broad, too narrow, or sufficiently defined.

## Narrowing Options
If the topic is too broad, narrow by:
- disease stage,
- population,
- intervention / exposure class,
- evidence type,
- method layer,
- clinical versus mechanistic focus.

## Output Expectation
The mapper should explicitly state what is included and what is excluded from the map.
