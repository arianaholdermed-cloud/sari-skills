# Finding Type Taxonomy

Classify the dominant current finding before selecting a validation route.

## 1. Descriptive Association Finding
Use when the current result is primarily correlative, observational, enrichment-based, or pattern-based without stronger mechanistic support.

## 2. Repeated Association Finding
Use when the association has been seen more than once, across datasets, cohorts, or settings, but still remains mainly associative.

## 3. Pathway / Activity Implication
Use when the core finding is a pathway-level signal, activity score, or inferred process implication.

## 4. Cell-State or Cell-Population Mechanism Signal
Use when the finding centers on a cell state, cellular subset, transition, or context-specific population pattern.

## 5. Target / Pathway Nomination Finding
Use when the result suggests a target, pathway, or intervention-relevant mechanism but does not yet establish actionability.

## 6. Biomarker-Mechanism Bridge Finding
Use when a measurable marker is linked to a biological process but the link remains incomplete.

## 7. Perturbation-Supported but Incomplete Mechanism Finding
Use when some functional support already exists, but specificity, context robustness, or patient-level relevance remains weak.

## 8. Clinical-Mechanistic Bridge Finding
Use when the finding already includes some patient-level link, but the bridge between biology and clinical use remains incomplete.

## Rule
If multiple types are present, identify:
- dominant finding type
- secondary finding type(s)
- what the mixed finding profile changes in the validation route
