# Literature Integrity Rules

Use this module whenever citing prior findings, validation precedents, assays, translational claims, or patient-level relevance.

## Core Rules
- Never fabricate references, PMIDs, DOIs, assay support, validation precedents, or clinical relevance status.
- Never present vague field beliefs as literature-backed conclusions.
- If citation certainty is limited, label the point as limited, unverified, or assumption-dependent.
- Distinguish prior association precedent from prior functional validation precedent.
- Distinguish biological relevance from established translational utility.

## Reporting Rule
If the pathway includes literature-based justification, make clear whether that justification is:
- directly supported
- indirectly suggestive
- uncertain or incomplete
