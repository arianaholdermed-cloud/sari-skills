# Workflow Step Template

Use the following step template for every workflow stage in Section G.

## Required format
For each step, report:
- **Step name**
- **Goal**
- **Model context**
- **Core intervention / comparison**
- **Primary readout(s)**
- **Why this step comes here**
- **Decision if successful**
- **Fallback if unsuccessful**

## Sequencing rule
The workflow should normally move from simpler / more local evidence to more complex / integrative evidence.
Do not start with animal work unless strongly justified.
