# Readout and Control Library

## Readout layers
Use a proximal + distal structure whenever possible.

### Proximal readouts
- target expression or perturbation confirmation
- pathway activation/inhibition markers
- immediate biochemical or molecular response

### Distal readouts
- proliferation / apoptosis / migration / invasion / differentiation
- secretion or functional phenotype
- in vivo growth / survival / burden / tissue phenotype

## Control logic
Typical control families include:
- untreated / vehicle control
- non-targeting perturbation control
- empty-vector or mock control
- positive control if a known direction exists
- rescue or reversal control when specificity matters
- batch / plate / litter-aware controls when relevant

## Important rule
Do not list controls generically. Tie each control to a specific interpretive need.
