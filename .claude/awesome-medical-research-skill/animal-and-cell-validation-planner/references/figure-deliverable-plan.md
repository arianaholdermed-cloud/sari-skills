# Figure and Deliverable Plan

Translate the design into an output package rather than only an experiment list.

## Typical figure logic
- Figure 1: claim context and baseline characterization
- Figure 2: perturbation confirmation and primary phenotype readout
- Figure 3: mechanism-support or rescue logic
- Figure 4: animal or orthogonal validation if included
- Figure 5: translational-support linkage back to the original human finding

## Deliverable rule
The final plan should specify the minimum useful output package, such as:
- one internal validation memo,
- one thesis-ready figure set,
- one manuscript-oriented figure skeleton.
