# Assumption Quality Audit

## Core Rule
Every major planning parameter must be labeled by evidence quality.

## Allowed Labels
- Known / directly provided
- Literature-supported but uncertain
- Locally estimated / institution-specific
- Guessed / speculative
- Missing and critical

## Parameters to Audit
Depending on study type, audit:
- event rate,
- outcome variance,
- effect size,
- dropout / loss to follow-up,
- exposure prevalence,
- allocation ratio,
- incidence rate,
- number of candidate predictors,
- expected number of events,
- subgroup size assumptions,
- external validation sample characteristics.

## Important Rule
Do not treat a guessed value as equivalent to an empirically supported value.

## Reporting Rule
If multiple critical parameters are guessed or missing, downgrade the recommendation from exact planning to range-based or feasibility-based planning.
