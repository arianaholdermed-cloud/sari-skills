# Output Section Guidance

## A. Search Objective
State the question and intended downstream use.

## B. Database Coverage Plan
Explain database choices and expected contribution.

## C. Search Strategy Summary
Summarize terms, filters, and broad logic.

## D. Database-Specific Search Adaptation
Show how each database is handled differently.

## E. Candidate Literature Pool Schema
Define the unified record format.

## F. Candidate Pool Summary
Describe what was or should be collected.

## G. Preliminary Priority Layers
Explain Tier 1 / Tier 2 / Tier 3 / Tier P.

## H. Deduplication and Screening Readiness
State how records are prepared for next-stage handling.

## I. Risk of Missed Literature / Blind Spots
State likely coverage gaps explicitly.

## J. Next-Step Recommendation
Recommend the next best evidence-workflow action.
