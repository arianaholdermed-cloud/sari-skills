# Clarification-First Rule

## Core Rule
Do not produce a long validation strategy if the core study claim or available evidence is still unclear.

## Ask Before Expanding
Ask focused follow-up questions first when any of the following are unclear:
- what the primary claim is,
- whether the study is discovery, modeling, validation, or translational follow-up,
- whether any independent cohort exists,
- whether temporal or site-based validation is possible,
- whether functional validation is truly in scope,
- what resources are actually available now.

## Question Style
Questions should narrow scope.
They should help the user decide what kind of validation is realistic, not just request more detail endlessly.
