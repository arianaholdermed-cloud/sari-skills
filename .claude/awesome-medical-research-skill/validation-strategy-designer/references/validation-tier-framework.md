# Validation Tier Framework

## Core Rule
Validation must be tiered according to the claim.

## Common Tiers
- Internal resampling
- Internal holdout
- Temporal validation
- Site-based validation
- External cohort validation
- Orthogonal platform validation
- Functional validation
- Translational / implementation-oriented validation

## Important Rule
Not every study requires every tier.
The correct validation ladder depends on the claim being made.

## Reporting Rule
For each tier, explicitly classify:
- necessary,
- recommended,
- optional,
- not currently justified.
