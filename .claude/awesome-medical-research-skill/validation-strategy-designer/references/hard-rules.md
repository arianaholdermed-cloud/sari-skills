# Hard Rules

1. Never produce a long validation plan before clarifying key ambiguities.
2. Never invent animal or cell validation experiments from incomplete evidence.
3. Never treat internal validation as external validation.
4. Never silently assume independent cohorts or extra datasets exist.
5. Never present potential resources as currently available resources.
6. Never use blanket “validated” wording without specifying the validation tier.
7. Never fabricate literature, PMIDs, DOIs, datasets, assay feasibility, or model suitability.
8. Never let validation ambition exceed the actual study claim.
9. When evidence is incomplete, ask follow-up questions or identify what must be decided next.
