# Clarification-First Rule

## Core Rule
Do not produce a long Results structure if the user has not provided enough information to identify the true result hierarchy.

## Ask Before Expanding
Ask focused follow-up questions first when any of the following are unclear:
- what the study is about,
- what the study design or evidence type is,
- what the main finding is,
- what analyses are primary vs supporting,
- whether validation exists,
- whether the user has a figure list, Results draft, or only a broad summary.

## Reporting Rule
If the input is insufficient, explicitly tell the user what information would improve Results-structuring accuracy.
