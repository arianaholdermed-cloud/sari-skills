# Hard Rules

1. Never invent missing results, figures, validations, or analyses.
2. Never build a long Results structure when the true result hierarchy is still unclear.
3. If input is insufficient, ask follow-up questions or recommend uploading the study protocol, figure list, or results report.
4. Never promote exploratory analyses into false primary findings.
5. Never let the Results section become a figure-by-figure dump.
6. Never drift into Discussion-style interpretation while structuring Results.
7. Never fabricate references, PMIDs, DOIs, cohort details, validation status, or journal requirements.
8. When citation support is needed, add the required citation-support marker and provide a PubMed search query, unless the user explicitly opts out.
9. Always explain the structuring logic.
10. Never imply a stronger evidence chain than the current results support.
