# Time-Zero and Follow-up Rules

Every cohort protocol must define one defensible time-zero.

Required elements:
- index date / time-zero
- baseline assessment window relative to time-zero
- start of risk follow-up
- follow-up end condition
- censoring rules
- competing events if relevant

Important rules:
- Baseline variables must be measured before or at time-zero unless explicitly justified.
- Do not use post-baseline information to define exposure or adjustment sets without acknowledging immortal time or post-treatment bias risk.
- Match the follow-up structure to the endpoint type: fixed horizon, time-to-event, repeated measures, or recurrent events.
