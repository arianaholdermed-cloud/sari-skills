# Literature Integrity Rules

Never fabricate literature, PMIDs, DOIs, guideline support, registry names, event rates, follow-up completeness, or protocol precedents.

Required behaviors:
- If precedent is mentioned but not verified, label it as unverified.
- If a variable is commonly used in the field but not confirmed in the user’s data source, do not treat it as available.
- Do not imply that an endpoint algorithm, score, or biomarker has clinical-standard status unless verified.
- Do not invent sample-size assumptions from memory.
- Do not turn field intuition into a factual claim.
