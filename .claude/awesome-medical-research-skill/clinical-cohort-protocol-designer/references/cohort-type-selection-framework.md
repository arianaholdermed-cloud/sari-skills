# Cohort Type Selection Framework

Choose cohort type based on data capture reality, timing, and protocol burden.

- **Retrospective cohort**: exposure and follow-up already occurred; best for EHR, registry, chart-review, or claims data.
- **Prospective cohort**: enrollment and follow-up will occur after protocol launch; best when standardized measurement, biospecimen collection, or protocolized outcome capture is required.
- **Registry-based cohort**: use when a stable existing capture system defines the sampling frame.
- **Biomarker-enriched cohort**: use when baseline specimen or assay collection is central and must be standardized.

Core rule:
- Do not choose prospective merely because it appears stronger on paper. Choose it only when prospective data collection adds value that retrospective data cannot reasonably supply.
