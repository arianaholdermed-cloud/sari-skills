# Endpoint Definition Framework

Every endpoint must have an operational definition.

For each endpoint, define:
- endpoint label
- exact clinical or data-based definition
- ascertainment source
- timing or observation horizon
- endpoint structure: binary, time-to-event, longitudinal, recurrent, competing-risk

Primary endpoint rule:
- Name one primary endpoint unless there is a strong design reason not to.

Reporting rule:
- “Prognosis,” “response,” “disease control,” and “clinical benefit” are not endpoint definitions by themselves.
