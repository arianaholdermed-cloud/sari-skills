# Method-Need Taxonomy

Use this module to classify the project before selecting algorithms.

## Core categories

### 1. Predictive biomarker / risk modeling
Use when the project predicts response, diagnosis, survival, progression, or classification labels.

### 2. Pathway-guided interpretable modeling
Use when pathway priors, mechanistic interpretability, or biologically grouped features are central.

### 3. Multi-omics integration
Use when multiple molecular layers must be integrated into a shared model.

### 4. Disease-gene / drug-target / network prioritization
Use when the project ranks genes, pathways, targets, or compounds.

### 5. Cell-state / spatial resolution
Use when the project needs cellular localization, deconvolution, trajectory, or spatial context.

### 6. Causality / mechanism upgrade
Use when predictive performance alone is insufficient and the project needs stronger causal or mechanistic support.

### 7. Translational utility / clinical stratification
Use when patient subgrouping, implementable stratification, or responder identification is the main goal.

## Rule
Classify by the **scientific problem first**, not by whatever algorithm is currently fashionable.
