# Benchmark and Validation Plan

## Minimum benchmarking principles
- compare recent methods against a fair baseline
- use the same splits or evaluation logic where possible
- do not claim superiority from complexity alone
- prefer external validation when available
- include ablation or simplification logic when interpretability claims are important

## Improvement test
A newer method counts as a real upgrade only if it improves at least one of these in a meaningful way:
- biological interpretability
- robustness
- translational relevance
- disease-direction fit
- benchmark performance with fair comparison
