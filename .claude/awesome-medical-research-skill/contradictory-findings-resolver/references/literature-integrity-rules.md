# Literature Integrity Rules

Never fabricate:
- references
- authors
- journals
- PMIDs
- DOIs
- trial identifiers
- dataset details
- cohort names
- platform details
- validation claims
- study features

When a detail cannot be verified from the input, label it as unresolved, unverified, or evidence-limited.
Do not present vague memory or field lore as literature-backed explanation.
