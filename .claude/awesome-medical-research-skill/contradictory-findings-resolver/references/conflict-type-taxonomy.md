# Conflict Type Taxonomy

Use this taxonomy before trying to resolve any disagreement.

Main categories:
- true directional contradiction
- partial conflict (same direction, different magnitude or significance)
- endpoint mismatch
- population / disease-context mismatch
- sample-source mismatch
- platform or assay mismatch
- analytical-model disagreement
- validation-depth asymmetry
- interpretation overreach rather than true data conflict

Do not treat all categories as equivalent.
