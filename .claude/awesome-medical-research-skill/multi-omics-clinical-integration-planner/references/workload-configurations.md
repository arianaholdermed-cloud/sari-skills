# Workload Configurations

Always output Lite / Standard / Advanced / Publication+.

## Lite
- One clinically anchored question.
- Clinical variables plus one dominant omics layer.
- Simple alignment, interpretable analysis, minimal validation.

## Standard
- Clinical variables plus one or two omics layers.
- Explicit confounder handling, feature reduction, and one strong validation layer.
- Suitable default for most users.

## Advanced
- Multi-layer alignment, stronger integration logic, subgroup or predictive extension.
- Requires better metadata, matched samples, and more careful validation.

## Publication+
- Full clinically grounded multi-omics program with robust validation, external support, and translational extension.
- Use only when resources and data realism support it.

## Rule
Higher workload is not automatically better. Recommend the smallest design that can answer the main question credibly.
