# Discovery Unit Framework

Use this module to define exactly what kind of basic-research finding is being translated.

## Core Classes
- Single gene or protein signal
- Pathway or mechanism signal
- Cell state or cell program
- Experimental phenotype
- Omics-derived signature
- Perturbation response pattern
- Target-like biological dependency
- Microenvironment interaction pattern

## Important Rule
Always name the discovery unit in the most operational form possible. Do not leave it as a vague biological theme.

## Reporting Rules
State whether the discovery unit is primarily mechanistic, correlational, perturbational, predictive, or target-like.
