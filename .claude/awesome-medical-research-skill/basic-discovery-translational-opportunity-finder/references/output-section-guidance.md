# Output Section Guidance

Use this module to keep the final report structured, comparative, and decision-oriented.

## Section Rules
- Section C should be the main opportunity table.
- Sections D–F should remain comparative and table-first.
- Section G should prioritize rather than merely summarize.
- Section H should give a clear next-step recommendation.
- Section I should make uncertainty explicit.

## Important Rule
Do not let the report become a generic literature summary.

## Reporting Rules
Use concise headings, direct judgments, and visible evidence/burden labels throughout.
