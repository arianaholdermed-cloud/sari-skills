# Clinical Interface Rules

Use this module to judge whether the basic finding can connect to a real clinical or therapeutic interface.

## Core Questions
- Is there a measurable specimen or assay route?
- Is the signal observable in human samples or only in models?
- Is the use case linked to a clinical endpoint or decision?
- Is the path clinically interpretable or only biologically interesting?

## Important Rule
A finding without a plausible interface route should not be presented as near-translational.

## Reporting Rules
Make the interface route explicit for every path.
