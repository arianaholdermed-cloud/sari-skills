# Conflict and Inconsistency Rules

Use this module whenever studies disagree.

Common conflict sources:
- endpoint mismatch
- treatment-setting mismatch
- platform/assay mismatch
- specimen timing mismatch
- population/subtype imbalance
- cut-point instability
- overfit signatures with poor transferability
- publication bias toward positive discovery studies

Reporting rules:
- Describe the conflict directly.
- Do not average contradictory evidence into a false neutral conclusion.
- If inconsistency is driven by setting differences, say which settings appear compatible and which do not.
