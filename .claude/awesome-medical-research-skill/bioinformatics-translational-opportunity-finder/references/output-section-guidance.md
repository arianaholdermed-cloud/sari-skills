# Output Section Guidance

Formatting standard for Sections A–I:
- Keep each section explicitly labeled.
- Use compact, decision-useful wording.
- Use a table only when parallel comparison materially improves clarity.
- Section C should distinguish plausible from premature translational paths.
- Section F must name one best-fit translational framing, not a vague list.
- Section G must show the reframing logic directly.
- Section I must include self-critical review items even when the translational case looks strong.
