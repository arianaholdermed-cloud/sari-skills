# Translation Barrier Rules

Use this module whenever a translational framing looks attractive but may be premature.

Common barriers:
- cohort specificity
- platform dependence
- unstable features or signatures
- weak biological anchoring for the proposed use case
- missing comparator evidence
- impractical assay route
- unclear workflow slot
- endpoint mismatch
- internal-validation-only support
- interpretation overreach

Reporting rules:
- Name the barrier directly.
- Do not soften a major barrier into vague caution language.
- When a mechanism-first interpretation is safer, say so explicitly.
