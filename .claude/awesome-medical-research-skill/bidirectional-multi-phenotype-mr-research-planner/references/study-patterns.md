# Study Patterns — Detailed Logic
# bidirectional-multi-phenotype-mr-research-planner

---

## Pattern A — Single-Family Bidirectional MR

**Use when:** User wants one disease/trait family tested against another disease/trait family in both directions.

## Pattern B — Multi-Phenotype Screening MR

**Use when:** User wants many exposures or many outcomes screened systematically.

## Pattern C — Subtype-Resolved MR

**Use when:** User wants major outcome subtypes or etiologic subtypes handled separately.

## Pattern D — Phenome-Style Bidirectional Causal Map

**Use when:** User wants broad bidirectional mapping across many trait pairs.

## Pattern E — Mechanism-Prioritized MR Follow-Up

**Use when:** User wants robust hits filtered for downstream biological interpretation or validation priority.

---

## Pattern Combination Rules

| Combination | When Appropriate |
|---|---|
| A + C | Bidirectional MR with explicit subtype resolution |
| B + C | Multi-phenotype screening with subtype-separated outcomes |
| D + E | Broad bidirectional map followed by robust-hit prioritization |
| A + E | One family-vs-family bidirectional MR plus shortlist for downstream validation |
| B + E | Multi-pair screening followed by follow-up ranking |
