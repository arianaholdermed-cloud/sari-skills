# Format Routing Rules

## Core Rule
Choose the output format that best matches the user’s need and the current execution environment.

## Available Routes
- Direct graphical abstract generation
- Image-generation prompt
- Mermaid flowchart
- Designer-facing handoff brief
- Narrative copy for a graphical abstract

## Routing Logic
Use:
- direct graphical abstract generation when image capability is available,
- image-generation prompt when an AI drawing workflow is intended but direct rendering is not available,
- Mermaid when process flow is central,
- designer handoff when human visual execution is expected,
- narrative copy when wording is the primary immediate need.

## Important Rule
Direct generation should be the preferred route when the environment supports it and the study input is sufficient.
