# Upload Recommendation Rule

## Core Rule
When the current chat input is too incomplete to recover the study storyline, recommend better source materials.

## Recommended Uploads
- study protocol,
- title and abstract,
- introduction logic,
- figure list,
- figure legends,
- results report,
- result summary table.

## Important Rule
This is not a refusal.
It is the most direct way to improve graphical-abstract accuracy.
