# Clarification-First Rule

## Core Rule
Do not produce a long graphical abstract narrative if the user has not provided enough information to identify the real study storyline.

## Ask Before Expanding
Ask focused follow-up questions first when any of the following are unclear:
- what the study is about,
- what the study design or evidence type is,
- what the workflow is,
- what the main finding is,
- what implication can actually be shown,
- whether the user wants a prompt, Mermaid flow, narrative copy, or designer brief.

## Reporting Rule
If the input is insufficient, explicitly tell the user what information would improve graphical-abstract accuracy.
