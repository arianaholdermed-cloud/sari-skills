# Prognostic Question Fit Rules

## Core Rule
Only treat the request as prognostic if the intended use is to estimate future outcome risk among people already in a defined population or disease state.

## Distinguish From Nearby Tasks
- Prognostic ≠ diagnostic classification at baseline.
- Prognostic ≠ treatment-response prediction.
- Prognostic ≠ mechanistic association without explicit follow-up outcome framing.

## Required Behaviors
- Name the intended prognostic use explicitly.
- State the outcome family.
- State what nearby biomarker-use families were considered and excluded.
