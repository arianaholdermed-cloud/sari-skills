# Candidate Marker and Screening Rules

## Core Rule
Marker generation, preselection, and confirmatory use must be separated.

## Required Behaviors
- State whether the marker set is pre-specified, moderately exploratory, or high-dimensional exploratory.
- State the rationale level for candidate inclusion.
- Distinguish forced clinical covariates from exploratory biomarker features.
- Mark exploratory features as exploratory.

## Important Rule
Do not present broad-screen feature selection as if it were already validated marker selection.
