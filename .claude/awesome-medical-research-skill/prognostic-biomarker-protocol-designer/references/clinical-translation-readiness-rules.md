# Clinical Translation Readiness Rules

## Core Rule
Clinical translation readiness must be stated honestly and stage-appropriately.

## Required Behaviors
- State whether the work is discovery-stage, development-stage, or validation-ready.
- State assay and implementation realism.
- State whether incremental clinical value is demonstrated or only hypothesized.
- State the next validation threshold needed before stronger translational claims.

## Important Rule
Do not call a biomarker translationally ready just because it is statistically associated with outcome.
