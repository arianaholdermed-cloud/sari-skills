# Workflow Step Template

## Core Rule
Follow the step order explicitly.

## Required Sequence
1. Clarify prognostic use.
2. Define cohort backbone.
3. Define endpoint and time scale.
4. Select study family.
5. Define candidate-marker strategy.
6. Build model-development line.
7. Define validation architecture.
8. Audit overfitting and leakage.
9. Review translation readiness.
10. Recommend the lead protocol version.
