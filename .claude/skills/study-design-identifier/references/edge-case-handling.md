# Edge-Case Handling

## Frequent edge cases
- A retrospective hospital database paper called “real-world”
- A prognostic signature paper described as “mechanism” because it includes limited wet-lab checks
- A case-control paper mislabeled as cross-sectional
- A cohort paper with only one biomarker measurement but longitudinal outcomes
- A TCGA/GEO screening study plus small-qPCR validation presented as a clinical biomarker validation study

## Resolution rule
When in doubt, separate:
- main design
- validation layer
- evidence-family position
- confidence level

Prefer an honest mixed classification over a falsely neat label.
