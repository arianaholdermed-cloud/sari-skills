# Feasibility Burden Framework

Assess feasibility under real constraints, not ideal conditions.

Required dimensions:
- data or sample access,
- preprocessing and annotation burden,
- method complexity,
- computational burden,
- assay or experimental burden,
- validation burden,
- collaborator dependence,
- timeline burden,
- failure sensitivity.

For each dimension, state:
- current condition,
- main bottleneck,
- whether the bottleneck is removable,
- whether the burden belongs to phase 1 or can be deferred.
