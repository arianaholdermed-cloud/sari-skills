# Validation Depth Framework

Use this module to classify how far a field has moved beyond discovery-only evidence.

## Validation Levels
- **Discovery only**
- **Internal-only validation**
- **External validation**
- **Orthogonal validation**
- **Cross-setting / cross-platform replication**
- **Implementation-relevant validation**

## Reporting Rules
Always state what kind of validation is missing. Do not use “validated” without specifying the level.
