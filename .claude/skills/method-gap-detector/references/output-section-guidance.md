# Output Section Guidance

Use this module to keep the report concise, structured, and decision-oriented.

## Section Priorities
- define the topic precisely,
- classify the main gap types clearly,
- separate consequential from minor weaknesses,
- recommend an upgrade path that fits the detected gap,
- state uncertainty where evidence is thin.

## Important Rule
The report should help the user decide what methodological weakness to fix next, not merely restate that the literature has limitations.
