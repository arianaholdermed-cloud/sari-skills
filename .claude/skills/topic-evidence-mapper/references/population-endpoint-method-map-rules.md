# Population, Endpoint, and Method Map Rules

## Population / Setting Layer
Map the main populations, disease stages, sample sources, model systems, and care settings.

## Endpoint Layer
Map common outcomes such as:
- survival,
- response,
- progression,
- pathology severity,
- biomarker expression,
- pathway activity,
- functional phenotype.

## Method Layer
Map common method families such as:
- differential analysis,
- cohort statistics,
- machine learning,
- single-cell,
- spatial,
- network analysis,
- MR / causal inference,
- wet-lab validation.

## Rule
These three layers should not be merged into an undifferentiated summary.
