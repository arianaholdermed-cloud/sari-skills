# Entry-Point Suggestion Rules

## Purpose
Help the user decide where to enter the field after seeing the map.

## Entry-Point Types
Possible entry points include:
- mature dense area for fast onboarding,
- manageable sub-area for focused reading,
- underdeveloped but plausible sub-area for future gap analysis,
- method-rich area for computational entry,
- clinically grounded area for translational entry.

## Rule
Entry-point suggestions should be practical and map-based, not overconfident promises of novelty.
