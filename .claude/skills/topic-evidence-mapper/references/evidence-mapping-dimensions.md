# Evidence Mapping Dimensions

## Core Dimensions
The evidence map should default to these dimensions:
- research streams,
- population / setting,
- endpoints,
- methods,
- evidence types,
- density,
- thin areas.

## Optional Dimensions
Add only if helpful:
- time trend,
- comparator patterns,
- translational depth,
- data modality,
- validation layer.

## Rule
Do not collapse all dimensions into a single narrative paragraph.
