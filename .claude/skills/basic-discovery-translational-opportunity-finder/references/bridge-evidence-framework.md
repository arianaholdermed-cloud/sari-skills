# Bridge-Evidence Framework

Use this module to judge whether a basic finding has enough evidence to support a translational path.

## Bridge Levels
- Mechanism-only signal
- Mechanism plus disease association
- Human-linked biological signal
- Outcome-linked translational signal
- Repeatedly supported translational bridge

## Important Rule
Mechanism plausibility alone is not a translational bridge.

## Reporting Rules
For each path, state whether the bridge evidence is direct, partial, weak, or missing.
