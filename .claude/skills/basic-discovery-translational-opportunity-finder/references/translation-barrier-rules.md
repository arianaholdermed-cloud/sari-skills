# Translation Barrier Rules

Use this module to identify why a basic finding may fail to become a strong translational topic.

## Common Barrier Types
- Weak human bridge evidence
- Endpoint mismatch
- Unclear assay route
- Low reproducibility
- High implementation burden
- Crowded opportunity space
- Weak differentiation
- Overclaimed therapeutic relevance

## Important Rule
Do not recommend a path without naming the main barrier.

## Reporting Rules
Each prioritized path should include one main barrier and one main missing proof requirement.
