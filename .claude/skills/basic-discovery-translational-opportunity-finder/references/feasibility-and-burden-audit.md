# Feasibility and Burden Audit

Use this module to compare whether translational paths are realistic to build.

## Core Burden Dimensions
- Assay burden
- Sample access burden
- Method complexity
- Validation burden
- Timeline burden
- Dependency burden
- Implementation friction

## Important Rule
A path can be biologically attractive but still impractical.

## Reporting Rules
Score burden qualitatively and state the main driver of friction for each path.
