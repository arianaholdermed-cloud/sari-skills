# Translational Use-Case Framework

Use this module to classify the translational direction being proposed.

## Core Use Cases
- Diagnosis
- Disease stratification
- Prognosis
- Treatment-response prediction
- Resistance prediction
- Monitoring or recurrence surveillance
- Trial enrichment
- Target nomination
- Drug-combination rationale
- Therapeutic-development direction

## Important Rule
Do not mix biomarker-style use cases with therapeutic-development use cases without stating the difference.

## Reporting Rules
Each opportunity path should be labeled with one primary use case and, only if necessary, one secondary use case.
