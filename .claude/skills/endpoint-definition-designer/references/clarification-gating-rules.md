# Clarification Gating Rules

Ask narrowing questions before long-form drafting if any of the following are unclear:
- disease or condition
- study objective
- study type or use case
- intervention / exposure / biomarker context when relevant
- likely endpoint family
- time origin or follow-up horizon
- data-capture or adjudication constraints

Important Rule:
Do not generate a long endpoint framework when the request is too broad to support an operational definition. Ask first, then draft.
