# Study Patterns — Detailed Logic
# two-sample-mr-exposure-screening-reference-grounded-research-planner-aligned

---

## Pattern A — Single-Exposure Primary MR Workflow

**Use when:** User wants a focused test of one primary exposure against one outcome.

## Pattern B — Exposure-Panel Screening Workflow

**Use when:** User wants many related exposures screened under one shared outcome.

## Pattern C — Robustness and Sensitivity Workflow

**Use when:** User wants pleiotropy, heterogeneity, leave-one-out, and estimator coherence emphasized.

## Pattern D — Extended Causal Architecture Workflow

**Use when:** User wants reverse MR, MVMR, or directionality upgrades.

## Pattern E — Replication and Triangulation Workflow

**Use when:** User wants independent GWAS replication or non-MR supportive triangulation.

---

## Pattern Combination Rules

| Combination | When Appropriate |
|---|---|
| A + C | focused exposure test with reviewer-grade sensitivity |
| B + C | exposure screening plus robustness filtering |
| A + D | single-exposure paper with reverse MR or MVMR extension |
| B + E | top-hit screening paper with replication |
| A + C + E | strongest conventional two-sample MR paper logic |
