# Plan Route Taxonomy

Classify the serious candidate route families before comparing them.

## 1. Retrospective Clinical Cohort Route
Use when the question requires outcome-linked clinical association, prognostic assessment, diagnostic evaluation, or retrospective stratification.

## 2. Prospective Cohort / Observational Route
Use when temporality, forward follow-up, or stronger real-world outcome capture is central.

## 3. Public-Dataset Bioinformatics Route
Use when the project is still discovery-heavy, data-led, or hypothesis-generating and can reasonably start from existing datasets.

## 4. Multi-Omics Integration Route
Use when the key value depends on integrated molecular resolution rather than a single data modality.

## 5. Mechanism-First Wet-Lab Route
Use when the core uncertainty is functional relevance, biological mechanism, perturbation response, or causal pathway logic.

## 6. Translational Biomarker-Validation Route
Use when the discovery signal already exists and the key task is validation toward use-case readiness.

## 7. Real-World Evidence Route
Use when care heterogeneity, implementation reality, transportability, or external practice relevance is central.

## 8. Target / Pathway Validation Route
Use when the main question concerns tractability, pathway dependency, or target relevance rather than broad clinical association.

## 9. Hybrid Staged Route
Use when one route should clearly lead and another should follow as validation, extension, or mechanistic strengthening.

## 10. Exploratory Pilot Route
Use when the project is still too early or too resource-limited for a full-scale primary path.

## Rule
If multiple route families appear plausible, identify:
- strongest candidate route family
- meaningful alternatives
- route families that should be excluded early
