# Analysis Line Framework

Lead with one coherent primary analysis line.

Minimum structure:
- main estimand or primary comparison concept
- model family matched to endpoint structure
- primary adjustment strategy
- subgroup / interaction plan
- sensitivity analysis logic
- missing-data handling concept

Examples:
- Cox or flexible survival model for time-to-event outcomes
- logistic model for fixed-horizon binary outcomes
- mixed-effects model for repeated longitudinal measures
- Fine-Gray or cause-specific framework when competing risk is central

Do not present an unordered menu of models without naming the lead analysis.
