# Variable Collection Taxonomy

Organize variables by role in the protocol rather than as a flat list.

Recommended buckets:
- exposure / predictor of interest
- core baseline demographics
- disease characteristics and severity
- treatment history / concomitant treatment
- comorbidities
- laboratory / imaging / pathology
- healthcare utilization or process variables
- confounders
- effect modifiers
- post-baseline follow-up variables
- endpoint adjudication variables

Priority rule:
- Separate variables into necessary, recommended, and optional.
- Distinguish baseline variables from post-baseline variables explicitly.
