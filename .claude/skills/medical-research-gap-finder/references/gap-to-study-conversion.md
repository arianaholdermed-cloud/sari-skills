# Gap-to-Study Conversion
# medical-research-gap-finder

---

## Goal

This module converts only the strongest evidence-audited gaps into study-ready opportunities.

---

## Required Conversion Table

| Priority Gap | Best-Fit Research Style | Core Inputs Needed | Minimal Executable Version | Stronger Publishable Version |
|---|---|---|---|---|

---

## Conversion Rules

### Best-Fit Research Style
Choose the study style that most directly answers the gap:
- clinical observational
- retrospective biomarker study
- bulk omics reanalysis
- single-cell / spatial reanalysis
- MR / causal inference
- mechanistic cell / animal validation
- translational validation
- evidence synthesis / structured review

### Core Inputs Needed
State the minimum data or material needed.
Examples:
- public cohort(s)
- external validation cohort
- scRNA-seq dataset
- tissue samples
- cell line perturbation system
- exposure and outcome GWAS

### Minimal Executable Version
Describe the smallest coherent study that can answer the question without overclaiming.

### Stronger Publishable Version
Describe what additional module materially upgrades the credibility.
Examples:
- independent validation cohort
- orthogonal validation layer
- functional perturbation
- subgroup / treatment-context analysis
- prospective or multi-center expansion

---

## Recommendation Rule

The final Primary Recommended Direction should come from the gap with the best balance of:
- credibility
- specificity
- answerability
- likely scientific or translational value

Do not choose a recommendation merely because it sounds more technologically advanced.
