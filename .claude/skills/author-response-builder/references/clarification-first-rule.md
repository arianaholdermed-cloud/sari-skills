# Clarification-First Rule

## Core Rule
Do not produce a long author-response draft if the review package and revision status are still too incomplete.

## Ask Before Expanding
Ask focused follow-up questions first when any of the following are unclear:
- what the reviewer comments actually are,
- what changes have already been made,
- what changes were not feasible,
- whether there is an editor letter,
- whether the response should be grouped by reviewer or comment cluster.

## Reporting Rule
If the input is insufficient, explicitly tell the user what information would improve response-drafting accuracy.
