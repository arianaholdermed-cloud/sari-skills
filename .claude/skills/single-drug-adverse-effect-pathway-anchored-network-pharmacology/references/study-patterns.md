# Study Pattern Coverage Matrix
# single-drug-adverse-effect-pathway-anchored-network-pharmacology-research-planner

---

## Pattern A — Canonical Pathway-Anchored Workflow
Use when the paper should center on overlap targets, enrichment-guided pathway anchoring, core-target nomination, and docking.

## Pattern B — Electrophysiology / Excitability Workflow
Use when channel, conduction, excitability, or electrophysiologic pathway interpretation dominates the endpoint logic.

## Pattern C — Immune-Hematologic Workflow
Use when immune, inflammatory, or hematologic pathway logic provides the clearest mechanistic anchor.

## Pattern D — Organ-Toxicity Workflow
Use when organ-context pathways provide clearer interpretation than topology-first hub selection.

## Pattern E — Translational Validation Workflow
Use when the user wants a stronger orthogonal support layer after the core pathway-anchored story is built.
