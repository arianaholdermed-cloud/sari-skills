# Figure and Deliverable Plan
# single-drug adverse-effect network pharmacology planners

---

## Minimum Figure Set by Configuration

### Lite
1. Study design overview
2. Drug targets / adverse-effect targets / overlap Venn or flowchart
3. PPI or pathway overview figure
4. One enrichment or docking figure
5. Integrated conservative conclusion schematic

### Standard
1. Study design and data-source map
2. Overlap target figure with harmonization summary
3. PPI and target prioritization or pathway anchoring figure
4. GO / KEGG / Hallmark enrichment figure
5. Docking overview for nominated targets
6. Literature-supported mechanism synthesis figure
7. Optional orthogonal support figure

### Advanced
Add:
- multi-database robustness figure
- structure-quality or target-confidence filter figure
- transcriptomic or literature triangulation figure

### Publication+
Add:
- orthogonal validation summary panel
- reviewer-facing evidence hierarchy figure
- optional pharmacovigilance coherence or wet-lab validation figure

---

## Mandatory Deliverables

- concise study abstract / design summary
- configuration comparison table
- dependency map / evidence map
- step-by-step workflow
- figure list with one-line purpose each
- validation and robustness section
- minimal executable version
- publication upgrade path
- reference literature pack
- self-critical risk review
