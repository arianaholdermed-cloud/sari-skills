# Literature Retrieval and Citation Standard
# single-drug adverse-effect network pharmacology planners

---

## Purpose

This literature module exists to support design choices, not to pad the output with a review.
Only retrieve references that help justify endpoint framing, drug rationale, method choice,
or comparable precedent structure.

## Required Rules

- Never fabricate citations.
- Only output formal references that are directly verified.
- Every formal reference must include a DOI, PMID, PMCID, or stable public link.
- Prefer PubMed-indexed records, PMC, publisher pages, or other durable scholarly records.
- If a candidate paper cannot be verified, do not list it as a formal reference.
- If no directly verified reference is available, say so explicitly.
- If browsing is unavailable, output a search strategy plus evidence gaps rather than fake references.

## Required Retrieval Targets

1. 2–4 background references on the drug, endpoint, or adverse-effect biology
2. 1–2 method references for the actual modules used
3. 1–2 similar-study precedents using comparable network-pharmacology logic
4. 1 explicit evidence-gap note

## Preferred Output Format

- Core background references
- Method justification references
- Similar-study precedent references
- Search strategy and evidence gaps
