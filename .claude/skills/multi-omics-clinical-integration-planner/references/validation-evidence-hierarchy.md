# Validation Evidence Hierarchy

Separate these layers clearly:
1. **Discovery evidence** — initial cross-layer signal finding.
2. **Internal support** — resampling, held-out testing, stability checks, calibration checks.
3. **External support** — independent cohort or dataset.
4. **Orthogonal support** — another assay, protein-level support, targeted panel, or alternative measurement.
5. **Experimental / translational extension** — functional follow-up, assay development, or implementation testing.

## Rule
Improved performance in one integrated cohort is not enough to claim clinical readiness.
