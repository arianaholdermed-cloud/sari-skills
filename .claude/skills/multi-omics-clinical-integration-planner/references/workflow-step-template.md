# Workflow Step Template

Recommended sequence:
1. Define the clinical use case and cohort logic.
2. Define data layers and subject/timepoint alignment.
3. Audit metadata sufficiency and confounders.
4. Select the dominant omics layer(s) and reduction strategy.
5. Run discovery / association modules.
6. Build the integration or fusion layer.
7. Evaluate interpretation and incremental value over clinical-only baselines.
8. Apply internal validation.
9. Add external / orthogonal validation if available.
10. Define translational next step.

## Rule
If datasets or public resources are named in the workflow, insert the mandatory Dataset Disclaimer immediately before Step 1.
