# Literature Retrieval and Citation Rules

Only provide formal references when directly verified.

## Allowed uses
- design justification
- method precedent
- disease-context framing
- same-use-case benchmark studies

## Rules
- Never fabricate PMIDs, DOIs, titles, journals, or trial status.
- If verification is unavailable, say so and provide a search strategy instead.
- Do not present unverified literature memory as confirmed citation support.
