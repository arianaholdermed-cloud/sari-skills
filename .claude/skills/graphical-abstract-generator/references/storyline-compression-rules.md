# Storyline Compression Rules

## Core Rule
A graphical abstract should express the study’s main storyline, not reproduce the full paper.

## Compress Toward
- problem,
- workflow,
- primary finding,
- implication.

## Important Rule
Only retain supporting layers that are necessary to make the main story intelligible.

## Avoid
- full-method detail,
- every subgroup analysis,
- every secondary result,
- long textual labels,
- redundant arrows and boxes.

## Reporting Rule
Explain why the chosen storyline is the minimum defensible version of the study.
