# Logic Reporting Rule

## Core Rule
Every major graphical simplification choice should be explained.

## Explain In Terms Of
- clearer study storyline,
- better visual compression,
- reduced overload,
- improved workflow legibility,
- cleaner distinction between main and supporting layers,
- protection against graphical overclaiming.

## Important Rule
Do not say only that the figure is “clearer.”
State why the chosen abstraction improves scientific communication.
