# Direct Generation Priority Rules

## Core Rule
When direct image capability is available and the study input is sufficient, prioritize direct graphical abstract generation.

## Priority Order
1. Direct graphical abstract generation
2. Image-generation prompt
3. Mermaid flowchart
4. Designer-facing handoff brief
5. Narrative copy only

## Important Rule
Do not hide or understate direct generation capability when it exists.

## Boundary Rule
Do not claim direct generation is available when the environment does not support it.
