# Confounder Identification Rules

## Core Rule
A plausible confounder is a variable that can distort the target association because it is related to both the exposure and the outcome or meaningfully structures baseline comparability.

## Use These Prompts
Ask:
- Could this variable influence who receives or exhibits the exposure?
- Could this variable independently affect the outcome?
- Is this variable measured before the exposure or index time?
- Is this variable a cause of the exposure, a cause of the outcome, or both?

## Important Rule
Do not call a variable a confounder just because it is clinically important, statistically significant, or strongly predictive.

## Reporting Rule
List only the most plausible confounders, not every available covariate.
