# Overadjustment and Collider Rules

## Core Rule
Conditioning on the wrong variable can create or amplify bias.

## Watch For
- mediators being treated as confounders
- post-baseline treatment response being treated as a baseline covariate
- severity variables measured after exposure being adjusted without justification
- variables affected by both exposure and outcome risk being conditioned on as if helpful

## Important Rule
A clinically intuitive variable may still be unsafe to adjust for.

## Reporting Rule
When you advise against adjustment, state whether the main concern is mediation, collider bias, post-baseline timing, or chronology uncertainty.
