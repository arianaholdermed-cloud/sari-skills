# Time Zero, Exposure, and Follow-Up Rules

## Core Rule
Every RWE design must define a single coherent time-zero logic tied to eligibility, baseline ascertainment, exposure assignment, and follow-up start.

## Required Elements
- Index date / time zero.
- Baseline lookback window.
- New-user versus prevalent-user framing.
- Exposure episode construction and grace periods if relevant.
- Outcome risk window and any induction / lag logic.
- Follow-up end rules.
- Censoring rules.
- Competing-event assumptions when relevant.

## Important Rule
Do not classify post-index variables as baseline confounders.

## Important Rule
If treatment is defined after a waiting period or accumulated exposure period, check for immortal time bias.
