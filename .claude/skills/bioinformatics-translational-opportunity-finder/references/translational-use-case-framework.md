# Translational Use-Case Framework

Use this module to assign the strongest plausible translational framing.

Allowed use-case categories:
- Diagnosis / detection
- Disease stratification / subtype assignment
- Prognosis / survival / progression risk
- Treatment-response prediction
- Monitoring / recurrence surveillance
- Target or pathway nomination
- Biology-guided enrichment hypothesis
- Mechanism-first follow-up when direct translation is still premature

Rules:
- Prognostic does not equal predictive.
- Target nomination does not equal tractable drug-development opportunity.
- Monitoring requires more than cross-sectional association.
- Diagnosis requires realistic detectability and comparator context.
- Use a narrower category when evidence is thin.
