# Bridge Evidence Framework

Use this module to assess what evidence connects the bioinformatics finding to a real translational application.

Bridge evidence may include:
- independent cohort support
- cross-platform consistency
- orthogonal validation with non-computational evidence
- endpoint alignment
- comparator analysis against current standards
- specimen accessibility support
- biological anchoring strong enough for the claimed use case
- workflow relevance beyond exploratory association

Reporting rules:
- Separate currently available bridge evidence from missing bridge evidence.
- If the translational story depends on evidence not yet shown, say so directly.
- Thin bridge evidence should downgrade the framing, not be hidden inside optimistic language.
