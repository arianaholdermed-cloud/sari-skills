# Validation Burden Framework

Use this module to estimate how much work is required before a translational claim becomes credible.

Common burden dimensions:
- external cohort burden
- endpoint refinement burden
- assay transfer burden
- orthogonal validation burden
- wet-lab dependency burden
- prospective or real-world evidence burden
- comparator burden
- implementation-study burden

Reporting rules:
- Keep validation burden separate from publication novelty.
- High burden does not automatically kill a topic, but it should narrow the framing.
- If a path requires multiple unsolved dependencies, call it high-burden explicitly.
