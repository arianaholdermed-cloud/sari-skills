# Reframing Rules

Use this module to convert inflated translational claims into defensible topic framings.

Common reframing patterns:
- “diagnostic biomarker” → “candidate diagnostic signal requiring assay-transfer and external validation”
- “prognostic biomarker” → “prognostic candidate with limited generalization evidence”
- “treatment-response marker” → “response-associated hypothesis requiring matched validation cohorts”
- “therapeutic target” → “target/pathway nomination hypothesis”
- “precision medicine classifier” → “stratification model with unresolved transferability”

Important rule:
Prefer the narrowest framing that the evidence can honestly support.
