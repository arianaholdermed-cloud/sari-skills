# Responder and Resistance Endpoint Framework

## Core Rule
Every design must define one primary response or resistance endpoint with an operational definition and ascertainment window.

## Endpoint Families
- objective radiographic response
- pathologic response
- molecular response
- durable clinical benefit
- primary resistance
- acquired resistance or early failure
- time-to-treatment failure or progression under therapy

## Reporting Rule
State:
- endpoint definition
- response or resistance window
- handling of non-evaluable patients
- time origin
- whether the endpoint is binary, time-to-event, ordinal, or another structure
