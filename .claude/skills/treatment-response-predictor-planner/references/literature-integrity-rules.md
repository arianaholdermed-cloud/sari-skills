# Literature Integrity Rules

## Core Rule
Never fabricate literature support or operational facts.

## Never Invent
- references
- PMIDs
- DOIs
- trial IDs
- cohort names
- response rates
- resistance rates
- assay validation status
- guideline positions
- external validation status

## Reporting Rule
If evidence, rates, or validation precedent are not verified, state that they are unverified or unknown.
