# Analysis Modules

## Core Modules
1. **Question framing and design selection**
2. **GWAS and QTL data architecture review**
3. **Locus-definition and harmonization plan**
4. **Primary colocalization analysis**
5. **Candidate-gene / transcript / protein prioritization**
6. **Evidence-tier classification and claim-boundary review**

## Frequently Recommended Modules
- tissue / cell-type relevance review
- LD reference compatibility review
- annotation support around prioritized loci
- multi-QTL comparison within the same locus
- ambiguity review for multiple nearby genes or multiple signals

## Optional Advanced Modules
- fine-mapping
- conditional or multiple-signal decomposition
- SMR + HEIDI
- conventional MR after coloc support
- cell-type-specific or single-cell QTL extension
- regulatory annotation integration
- locus-to-gene triangulation with external biology support

## Inclusion Rules
- Every module must solve a specific design problem.
- Do not add optional modules unless they materially improve the answer.
- If the locus is structurally ambiguous, prioritize ambiguity control over adding decorative downstream analyses.
