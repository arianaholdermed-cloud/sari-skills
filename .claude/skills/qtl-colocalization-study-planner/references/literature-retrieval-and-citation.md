# Literature Retrieval and Citation Rules

## Core Principle
Literature support is optional unless the user asks for references or precedent framing. If literature is included, every citation must be real and directly verifiable.

## Required Citation Categories
When literature support is requested, organize references into:
1. background / disease-GWAS context
2. method justification (colocalization, fine-mapping, SMR, QTL interpretation)
3. similar-study precedents
4. unresolved verification needs

## Hard Rules
- Never invent PMIDs, DOIs, accession IDs, or consortia.
- Never cite a paper unless its relevance is clear.
- Never use references as decoration.
- If you cannot verify a citation, leave the slot uncited.

## Search Targets
Good literature targets include:
- disease-specific GWAS-to-gene interpretation studies
- eQTL/pQTL/sQTL colocalization methodology papers
- fine-mapping and multi-signal interpretation papers
- cell-type-specific eQTL colocalization studies when context matters
- studies that explicitly combine coloc with SMR or MR, while maintaining claim boundaries
