# Target Template Selection Rules

## Core Rule
The conversion must be anchored to the real target formatting framework, not guessed from prestige or field convention.

## Common Targets
- journal class
- conference template
- publisher submission style
- generic clean LaTeX manuscript structure

## Important Rule
When the exact target template is unavailable, say so clearly and fall back to a generic structured submission style rather than pretending precise compliance.

## Reporting Rule
State what target framework is confirmed and what remains uncertain.
