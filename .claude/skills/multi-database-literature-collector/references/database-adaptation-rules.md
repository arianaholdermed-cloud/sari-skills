# Database Adaptation Rules

Do not assume one search string can be copied across all databases unchanged.

## PubMed
- Use MeSH where appropriate, but do not rely on MeSH alone
- Combine controlled vocabulary with free text
- Adapt field tags carefully

## Google Scholar
- Broader and noisier; keep queries concise
- Use phrase searching where appropriate
- Scholar is useful for recall and citation discovery, not clean metadata alone

## Web of Science
- Good for indexed retrieval and citation-linked search
- Adapt field restrictions and phrase logic to WoS syntax

## Preprint Servers
- Use focused, narrower queries
- Explicitly label results as preprints

## Adaptation Output
For each database, describe:
- query style
- field focus
- date filters
- database-specific compromise
