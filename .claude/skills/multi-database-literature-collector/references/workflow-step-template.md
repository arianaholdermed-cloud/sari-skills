# Workflow Step Template

Use the following stepwise design when presenting the collection workflow.

## For Each Step, State
- objective
- required inputs
- search or organization action
- expected output
- dependency note

## Recommended Step Sequence
1. clarify objective
2. choose databases
3. construct core search logic
4. adapt search per database
5. normalize records
6. assign first-pass tiers
7. prepare for dedup/screening
