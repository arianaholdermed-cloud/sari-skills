# Preliminary Priority Layering

This is first-pass organization, not final inclusion.

## Required Tiers
- **Tier 1**: highly likely core papers directly relevant to the question
- **Tier 2**: possibly relevant or borderline papers that need screening
- **Tier 3**: background, context, or indirect support
- **Tier P**: preprints, separated regardless of likely relevance

## Tiering Logic
Use the following signals:
- directness to the question
- study type relevance
- recency when appropriate
- relevance of population / intervention / condition / mechanism
- whether the record looks like original evidence vs background context

## Important Restriction
Do not use tiering to claim inclusion/exclusion finality.
