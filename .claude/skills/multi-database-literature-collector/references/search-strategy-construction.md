# Search Strategy Construction

The collection phase should optimize recall without becoming unmanageably vague.

## Build Search Elements
Minimum elements as relevant:
- disease / condition terms
- intervention / exposure / biomarker / mechanism terms
- outcome or evidence-type terms
- modality or method terms when needed
- synonyms, abbreviations, spelling variants

## Search Logic Principles
- Start broad enough to avoid premature exclusion
- Add narrowing only when the topic is too diffuse
- Use date filters only when the user asks or the task is update-oriented
- Use study-type filters only when necessary

## Good Practice
Prefer a layered strategy:
1. broad core concept search
2. concept + evidence-type refinement
3. optional recent-update layer

## Required Transparency
Always summarize:
- key terms used
- synonym logic
- filters used
- restrictions not used
