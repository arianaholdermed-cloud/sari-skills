# Deduplication and Screening Readiness

A good collection output should be easy to deduplicate and screen.

## Must Preserve Fields Helpful for Deduplication
- title
- authors
- year
- DOI
- PMID
- journal / venue
- source database

## Screening-Ready Output Should Support
- title/abstract review
- study-type filtering
- source-specific backtracking
- preprint separation
- later full-text retrieval

## Required Next-Step Clarity
Always say what should happen next:
- deduplicate
- title/abstract screen
- separate preprints
- prioritize Tier 1 reading
