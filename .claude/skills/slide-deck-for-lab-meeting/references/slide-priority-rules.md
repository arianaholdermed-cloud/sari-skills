# Slide Priority Rules

## Core Rule
A strong lab-meeting deck balances only the content needed for the discussion goal.

## Common Priority Buckets
- background and question
- workflow or design
- current data
- interpretation
- open problems
- next steps

## Important Rule
Too much background can waste time. Too much raw data can bury the point.

## Reporting Rule
Explain why some categories receive more slide weight than others.
