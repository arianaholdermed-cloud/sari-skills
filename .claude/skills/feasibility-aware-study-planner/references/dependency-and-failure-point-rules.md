# Dependency and Failure-Point Rules

Use this module to identify what could still break the study after narrowing.

## Typical Failure Points
- missing key covariates
- fragile sample-size assumptions
- hidden preprocessing burden
- unavailable assay support
- external validation dependency
- uncertain collaboration support
- time-window mismatch
- follow-up insufficiency

## Important Rule
Surface the dependencies that could invalidate the whole study, not just minor inconveniences.

## Reporting Rule
Separate critical dependencies from manageable operational friction.
