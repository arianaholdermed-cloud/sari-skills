# Resource Clarification Rules

Use this module before final planning whenever the user's resource situation is incomplete, vague, or only partially stated.

## Clarify Three Buckets Every Time
1. **currently available resources**
2. **potentially obtainable resources**
3. **unavailable or unrealistic resources**

## Typical Clarification Targets
- datasets
- cohorts or samples
- follow-up variables
- wet-lab capacity
- assay platforms
- analytical skills
- software or workflow familiarity
- external validation access
- collaboration support
- budget flexibility
- timeline

## Important Rule
Do not assume that potentially obtainable resources are dependable.

## Question Rule
Ask only the shortest set of questions that most reduces feasibility uncertainty.

## Reporting Rule
If the user does not clarify, explicitly label the plan as provisional and assumption-dependent.
