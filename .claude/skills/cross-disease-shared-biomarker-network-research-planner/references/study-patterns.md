# Study Patterns — Detailed Logic
# cross-disease-shared-biomarker-network-research-planner

---

## Pattern A — Shared-DEG Discovery Workflow

**Use when:** User wants the common differentially expressed genes across two related diseases.

## Pattern B — Hub-Gene Prioritization Workflow

**Use when:** User wants PPI-based key genes or central biomarkers.

## Pattern C — Regulatory-Network Interpretation Workflow

**Use when:** User wants TF-gene / TF-miRNA / upstream-regulation analysis.

## Pattern D — Immune and Drug-Follow-Up Workflow

**Use when:** User wants immune infiltration or candidate-drug screening after hub-gene discovery.

## Pattern E — Bioinformatics + Validation Workflow

**Use when:** User wants public validation plus qRT-PCR, cell-line, or tissue confirmation.

---

## Pattern Combination Rules

| Combination | When Appropriate |
|---|---|
| A + B | shared-DEG discovery plus hub-gene prioritization |
| B + C | hub-gene study plus regulatory-network interpretation |
| B + D | hub-gene study plus immune/drug follow-up |
| B + E | hub-gene study plus public and experimental validation |
| A + B + D | shared biomarkers plus translational follow-up prioritization |
