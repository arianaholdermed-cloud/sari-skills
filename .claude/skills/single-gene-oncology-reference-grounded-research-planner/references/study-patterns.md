# Study Patterns — Detailed Logic
# single-gene-oncology-reference-grounded-research-planner-aligned

---

## Pattern A — Expression and Differential-Context Workflow

**Use when:** User wants tumor-vs-normal expression or subgroup-expression support around a fixed target gene.

## Pattern B — Prognosis and Clinicopathologic Workflow

**Use when:** User wants survival endpoints, clinic association, or outcome-linked biomarker framing.

## Pattern C — Functional and Immune Interpretation Workflow

**Use when:** User wants pathway context, immune infiltration, or checkpoint association.

## Pattern D — Genomic / Epigenetic / Drug-Context Workflow

**Use when:** User wants CNV, mutation, methylation, proteomic, or drug-response context.

## Pattern E — Multi-Layer Public / Orthogonal Validation Workflow

**Use when:** User wants external cohort support, protein support, or cross-portal coherence.

---

## Pattern Combination Rules

| Combination | When Appropriate |
|---|---|
| A + B | baseline single-gene expression plus prognosis framing |
| A + C | expression support plus functional / immune interpretation |
| B + C | prognosis-centered paper with immune follow-up |
| B + D | prognosis-centered paper with genomic / epigenetic support |
| A + B + E | standard single-gene paper with external validation |
