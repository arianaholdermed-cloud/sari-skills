# Protocol Analysis Modules
# medical-research-gap-to-study-planner

---

## Module Families

### 1. Gap Clarification Modules
- gap decomposition
- scope bounding
- claim-boundary definition
- candidate-gap prioritization

### 2. Cohort / Data Modules
- discovery cohort selection
- external validation cohort selection
- subgroup or stage definition
- endpoint harmonization
- data-type appropriateness check

### 3. Biological / Statistical Modules
- differential analysis
- enrichment / pathway interpretation
- network prioritization
- deconvolution or cell-state projection
- subgroup heterogeneity assessment
- causal-support analysis
- response-stratification analysis

### 4. Validation Modules
- internal validation
- external validation
- orthogonal portal / protein support
- functional validation
- translational support

### 5. Interpretation Modules
- evidence-tier labeling
- limitation statements
- claim-boundary enforcement
- integrated evidence synthesis

---

## Module Selection Rule

Choose modules because they directly reduce the targeted uncertainty.
Do not add modules just because they are common in the literature.

Examples:
- a reproducibility gap usually needs stronger validation modules, not more enrichment plots
- a cell-state gap usually needs localization modules, not another prognostic model
- a translation gap usually needs clinically interpretable endpoint logic, not just pathway analysis
