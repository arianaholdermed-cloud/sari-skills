# Screening Value Rules

## Core Rule
Reading value depends on more than whether a paper is interesting.

## Common High-Value Features
- strong direct relevance,
- informative study design,
- adequate sample for the question,
- useful method detail,
- genuine novelty with plausible value,
- reproducibility or implementation relevance.

## Important Rule
A paper can have one of these without having all of them.

## Reporting Rule
State what specifically makes the paper high-value or low-yield for the user.
