# Clarification-First Rule

## Core Rule
Do not produce a long screening judgment if the paper information or reading goal is still too incomplete.

## Ask Before Expanding
Ask focused follow-up questions first when any of the following are unclear:
- what the paper is,
- what the user wants from it,
- whether the input is title only, abstract, or full text,
- whether the screening goal is relevance, methods, background, or benchmark value.

## Reporting Rule
If the input is insufficient, explicitly tell the user what information would improve screening accuracy.
