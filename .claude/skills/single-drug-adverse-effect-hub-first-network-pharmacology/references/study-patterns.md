# Study Pattern Coverage Matrix
# single-drug-adverse-effect-hub-first-network-pharmacology-research-planner

---

## Pattern A — Canonical Hub-First Workflow
Use when the paper should center on overlap targets, PPI topology, hub ranking, enrichment, and docking.

## Pattern B — Cardiotoxicity / Electrophysiology Workflow
Use when ion-channel, conduction, repolarization, or excitability biology dominates the endpoint logic.

## Pattern C — Immune-Inflammatory Workflow
Use when immune, inflammatory, or hematologic adverse-effect biology provides the clearest mechanistic anchor.

## Pattern D — Organ-Toxicity Workflow
Use when liver, kidney, neural, reproductive, or mixed-organ injury requires organ-context interpretation.

## Pattern E — Translational Validation Workflow
Use when the user wants a stronger orthogonal support layer after the core hub-first story is built.
