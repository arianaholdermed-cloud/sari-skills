# Current Claim Strength Rules

Use this module to judge what the current evidence already supports.

## Distinguish These Levels
- descriptive signal only
- repeated descriptive signal
- mechanistically suggestive
- functionally supported in part
- context-robust but mechanistically incomplete
- clinically linked but biologically incomplete

## Important Rule
Always write both:
- what the current evidence supports
- what the current evidence does not yet support

Do not let strong wording hide missing layers.

## Do Not Confuse
- replication with mechanism
- perturbation with specificity
- biomarker correlation with clinical utility
- pathway enrichment with causal proof
