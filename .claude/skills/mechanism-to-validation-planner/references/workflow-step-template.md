# Workflow Step Template

For each workflow step in the main output, include:

1. **Step Name**
2. **Objective**
3. **What This Resolves**
4. **What Is Already Supported vs Still Missing**
5. **Why It Comes Here in the Sequence**
6. **Expected Output**

## Typical Workflow Order
- interpret the current finding
- classify the finding type
- judge current claim strength
- identify the missing validation layer
- map candidate validation layers
- sequence the pathway
- separate necessary from optional steps
- define the clinical relevance bridge
- identify the weakest link
- recommend the primary route
