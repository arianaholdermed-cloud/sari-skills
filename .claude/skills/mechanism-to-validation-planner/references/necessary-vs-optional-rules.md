# Necessary vs Optional Rules

Use this module when deciding which validation steps must lead and which can be deferred.

## Necessary
Use for steps without which the main claim would remain too weak, too ambiguous, or too vulnerable to overclaim.

## Recommended
Use for steps that materially strengthen confidence, specificity, or transportability but are not always required for the first defensible pathway.

## Optional
Use for stronger, broader, or higher-burden extensions that add value but should not be treated as immediate requirements.

## Important Rule
Do not mark all stronger steps as necessary.

The primary route should remain disciplined.
