# Pathway Sequencing Rules

Use this module to order the validation ladder.

## Typical Ordering Logic
- strengthen weak association if the signal is unstable
- confirm orthogonally if platform dependence is a major concern
- move to perturbation when the main missing layer is function
- strengthen specificity when multiple explanations remain plausible
- extend into clinical relevance only after the biological ladder is strong enough

## Important Rule
Sequence steps by evidence dependency, not by prestige or technical intensity.

## Reporting Rule
For the primary pathway, state for each step:
- why it comes here
- what it resolves
- what would remain unresolved afterward
