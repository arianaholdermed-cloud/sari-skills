# Validation Layer Taxonomy

Use this module to classify the next possible validation layers.

## Common Validation Layers

### 1. Association Reinforcement
Use when the current signal needs stronger replication across datasets, contexts, or measurement settings.

### 2. Orthogonal Confirmation
Use when the same claimed phenomenon should be confirmed through a different assay, platform, modality, or measurement logic.

### 3. Functional Perturbation
Use when the core missing layer is whether changing the factor changes the phenotype, pathway, or system behavior.

### 4. Mechanistic Specificity
Use when the evidence lacks pathway dependency, mediator logic, rescue logic, or contrast against plausible alternatives.

### 5. Context Robustness
Use when the signal may be model-specific, platform-specific, lineage-specific, disease-stage-specific, or otherwise fragile.

### 6. Clinical Relevance Extension
Use when biological evidence is reasonable but the patient-level meaning, use-case positioning, or translational bridge remains weak.

## Reporting Rule
For each candidate layer, state:
- what it would test
- what uncertainty it would reduce
- whether it belongs early or later in the pathway
