# Researchability Assessment Rules

After clarification, assess whether the question is ready for evidence search or original study planning.

## Searchable
Can the question be translated into a reasonable literature retrieval task?

## Researchable
Can the question be studied using available clinical, observational, mechanistic, or omics evidence?

## Testable
Can the question be evaluated using a defined design, outcome, and evidence strategy?

## Typical Outcomes
- search-ready evidence question
- gap-finding question
- protocol-ready study question
- too broad and still needs narrowing
- too underspecified and needs additional user detail

## Important Distinction
A question can be searchable but not yet testable.
A question can be interesting but not yet bounded enough for study design.
