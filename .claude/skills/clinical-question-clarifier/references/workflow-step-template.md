# Workflow Step Template

For each workflow step in the main output, include:

1. **Step Name**
2. **Objective**
3. **What This Resolves**
4. **What Is Explicit vs Inferred**
5. **What Still Needs Narrowing**
6. **Expected Output**

## Typical Workflow Order
- interpret the user’s real intent
- classify question type
- identify ambiguity and missing elements
- choose best-fit framing structure
- define structured elements
- produce clarified question versions
- state scope boundaries
- recommend downstream next step
