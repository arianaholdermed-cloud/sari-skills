# Full Introduction Writing Rules

## Core Rule
A full Introduction should convert an already clear background-gap-objective logic into disciplined scientific prose.

## Write For
- coherent progression,
- controlled background density,
- clear gap narrowing,
- accurate study positioning,
- a clean objective statement.

## Avoid
- broad unfocused opening paragraphs,
- literature stacking,
- overclaiming,
- mini-review behavior,
- abrupt transition to the study objective.

## Important Rule
Do not attempt full-section writing before the underlying logic is sufficiently clear.
