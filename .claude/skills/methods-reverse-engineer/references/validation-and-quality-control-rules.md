# Validation and Quality Control Rules

Identify whether the paper includes reliability logic beyond the main analysis.

## Possible items
- sample QC
- assay QC
- internal validation
- external validation
- replication cohort
- wet-lab confirmation
- positive / negative / matched controls
- robustness / sensitivity / ablation analysis

## Rules
1. Separate QC from validation.
2. Separate internal validation from external validation.
3. Absence of validation should be stated explicitly.
