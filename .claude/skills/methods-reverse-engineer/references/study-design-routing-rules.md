# Study Design Routing Rules

Classify the actual method structure, not just paper labels.

## Common design families
- randomized or non-randomized interventional study
- cohort study
- case-control study
- cross-sectional study
- registry / real-world / claims / EHR study
- diagnostic accuracy study
- prognostic / predictive model study
- omics / computational discovery study
- mechanistic cell / animal / organoid experiment
- hybrid study combining two central evidence tracks

## Routing rules
1. Identify the dominant evidence-producing design.
2. Add a secondary family when another track is central to the main claim.
3. Do not label a study “omics design” when omics is only a measurement layer inside another design.
4. Use hybrid status when both tracks materially support the core conclusion.
