# Analysis Pipeline Reconstruction Rules

Reconstruct the analysis or experiment in sequence.

## Possible elements
- data cleaning / QC
- normalization / batch correction / transformation
- feature selection
- model fitting / statistical testing
- wet-lab intervention sequence
- image / signal / sequencing processing
- validation / sensitivity / subgroup analysis

## Rules
1. Keep sequence explicit.
2. Name the unit of analysis when possible.
3. Distinguish primary analysis from exploratory add-ons.
4. Do not assume package defaults or hidden analyst choices.
