# Data and Sample Extraction Rules

Extract the operational input structure.

## Capture when available
- data or specimen source
- study setting
- recruitment or acquisition route
- inclusion criteria
- exclusion criteria
- final analytic sample size
- subgroup definitions
- follow-up window or collection period
- train / test / validation split

## Rules
1. Distinguish recruited participants from analyzed samples.
2. Distinguish raw available sample size from final analytic size.
3. Mark sample flow as unclear if exclusions or losses are not fully reported.
