# Input Coverage and Boundary Rules

Use the most conservative coverage judgment that fits the provided material.

## Coverage levels
- **Level 1**: full methods available
- **Level 2**: partial methods / figures / supplements / selected sections only
- **Level 3**: title / abstract / DOI / PMID / citation only

## Rules
1. Never reconstruct hidden details as reported facts.
2. At Level 2, allow partial workflow reconstruction only where the text or figures strongly support it.
3. At Level 3, stop at design-level and high-level workflow likelihood. Do not provide full replication steps.
4. State the reconstruction boundary before the main output.
5. Mark unavailable details explicitly.
