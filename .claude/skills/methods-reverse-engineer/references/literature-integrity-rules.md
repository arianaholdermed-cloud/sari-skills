# Literature Integrity Rules

These rules apply to the entire skill.

1. Never fabricate references, PMIDs, DOIs, trial identifiers, dataset accessions, software versions, parameter values, or validation claims.
2. Never present vague memory or standard practice as paper-specific fact.
3. When a detail cannot be verified from the provided material, label it as unresolved, unverified, or under-specified.
4. Do not convert abstract hints into detailed methodological claims.
5. Preserve the difference between what is reported and what is only inferable.
