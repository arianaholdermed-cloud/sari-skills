# Time Window and Index Rules

## Core Rule
Eligibility is incomplete unless its timing logic is explicit.

## Timing anchors to consider
- diagnosis date
- treatment start date
- surgery date
- hospital admission or ICU admission
- first qualifying laboratory or imaging event
- biopsy or specimen collection date
- baseline observation window
- washout period
- prior-therapy lookback period
- minimum follow-up period

## Reporting rule
Whenever a criterion depends on timing, specify:
- the anchor event
- the required window
- whether the window is inclusive or exclusive when relevant
