# Inclusion Criteria Rules

## Core Rule
Inclusion criteria should define who belongs in the target population and why that membership can be verified.

## Good inclusion criteria usually specify
- disease confirmation method
- age requirement when relevant
- setting or care context when relevant
- stage / severity / subtype when central to the question
- treatment or exposure status when central to the study design
- specimen, imaging, pathology, or assay availability when necessary for the study objective
- minimum data completeness or follow-up requirement only if justified

## Avoid
- vague wording without operational anchors
- clinically appealing but unverifiable phrasing
- inclusion criteria that accidentally use post-baseline information unless the design truly requires retrospective classification
