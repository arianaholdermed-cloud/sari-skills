# Figure and Deliverable Plan

Choose figure logic that matches the story.

## Typical Core Figures
- study design and cohort grouping schematic
- QC / sample overview / batch structure figure
- primary contrast result figure
- pathway or functional interpretation figure
- subtype / score / clinical association figure when relevant

## Deliverable Rule
The output package should feel like a coherent study kit: main figures, supplementary checks, key tables, and a clear validation roadmap.
