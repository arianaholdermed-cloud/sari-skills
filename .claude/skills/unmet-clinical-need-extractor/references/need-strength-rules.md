# Need Strength Rules

Use this module to judge how strongly an unmet clinical need is supported.

The goal is not just to list candidate needs, but to classify how established each need actually is.

## Need Strength Classes

### Strongly Established
Use when the need is repeatedly acknowledged in guidelines, major reviews, and/or real-world practice evidence.

### Partially Supported
Use when the need is plausible and supported in multiple sources, but remains narrower, less consistent, or not universally established.

### Context-Dependent
Use when the need is real mainly in particular stages, populations, settings, or treatment contexts.

### Weakly Supported / Overstated
Use when the need mostly relies on generic burden language, technology enthusiasm, or limited-source overstatement.

## Important Rules

- Strongly established does not mean universally identical across all settings.
- Context-dependent needs can still be highly important if the context is clearly defined.
- Do not inflate a partially supported need just because it is fashionable.
- If evidence is indirect, make that visible.
