# Use-Case Framework

Assign the dominant intended use case.

Allowed use-case categories:
- Risk / susceptibility
- Early detection / diagnosis
- Differential diagnosis
- Disease stratification / subtype assignment
- Prognosis / survival / progression risk
- Predictive of treatment response
- Resistance prediction
- Monitoring / longitudinal disease activity
- Minimal residual disease / recurrence surveillance
- Toxicity / adverse event risk
- Mechanism-linked enrichment / biology-guided subgrouping

Rules:
- Prognostic ≠ predictive.
- Diagnostic ≠ screening/early-detection unless explicitly studied that way.
- A biomarker may have more than one proposed use case, but only assign multiple categories when the evidence truly supports multiple roles.
