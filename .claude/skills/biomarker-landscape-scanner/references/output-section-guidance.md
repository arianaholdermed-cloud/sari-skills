# Output Section Guidance

Formatting standard for Sections A–I:
- Keep each section explicitly labeled.
- Use compact, decision-useful wording.
- In Section C, organize by use case first whenever possible.
- Every mapped biomarker entry should include: name, type, specimen, use case, validation level, maturity label, limitation.
- Section D should foreground failure modes, not hide them.
- Section G must recommend one primary direction, not a vague basket of options.
- Section H must include self-critical review items, even when the field looks strong.
