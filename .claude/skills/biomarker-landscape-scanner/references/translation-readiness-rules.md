# Translation Readiness Rules

Assess practical deployment barriers.

Key dimensions:
- specimen accessibility
- assay complexity and cost
- turnaround time
- reproducibility across labs/platforms
- interpretability
- compatibility with existing workflow
- added value beyond current standard markers
- regulatory/implementation feasibility

Translation-ready does not mean “biologically interesting.”
A near-translation candidate should have repeated support, usable assay conditions, and a plausible path into real workflow.
