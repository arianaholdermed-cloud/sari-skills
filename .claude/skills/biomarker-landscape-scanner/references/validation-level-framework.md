# Validation Level Framework

Assign one primary validation level.

Recommended levels:
1. Discovery only
   - single cohort, exploratory association, no independent validation
2. Internal validation only
   - resampling, split-sample, or model-internal validation only
3. External retrospective validation
   - one or more independent retrospective cohorts or datasets
4. Strong multi-cohort support
   - repeated support across platforms/cohorts/settings
5. Prospective / implementation-oriented validation
   - prospective cohort, pragmatic validation, assay standardization, real-world deployment evidence
6. Clinical embedding / guideline-adjacent
   - formal role in guideline, consensus workflow, or established routine use

Do not jump levels without explicit evidence.
