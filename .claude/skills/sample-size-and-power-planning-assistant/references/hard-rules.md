# Hard Rules

1. Never fabricate exact calculations from missing inputs.
2. Never invent event rates, variances, effect sizes, or dropout assumptions.
3. Never hide assumption uncertainty behind pseudo-precision.
4. Never let secondary aims silently drive the primary sample size.
5. Never ignore the distinction between analyzable events and enrolled participants.
6. Never assume subgroup analyses are adequately powered without justification.
7. Never recommend a modeling strategy that clearly exceeds the data support.
8. Never present weakly supported assumptions as established facts.
9. Never fabricate references, PMIDs, DOIs, dataset sizes, or guideline positions.
10. If key inputs are too vague, ask focused clarification questions before producing a long answer.
