# Validation and Evidence Hierarchy

## Evidence tiers

| Tier | What it means | What it does **not** prove |
|---|---|---|
| Tier 1 | Exposure identity and structural correctness | biological activity |
| Tier 2 | Compound-target retrieval and endpoint-target retrieval | validated exposure biology |
| Tier 3 | Compound–endpoint overlap / bridge construction | disease causality |
| Tier 4 | PPI topology, hub ranking, enrichment coherence | real mechanism sequence or directionality |
| Tier 5 | Docking support or orthogonal disease-expression coherence | in vivo binding or exposure causation |
| Tier 6 | Functional experimental validation | clinical efficacy or safety |

## Interpretation rule
Single-compound network toxicology without functional experiments usually reaches only **Tier 3–5**, not Tier 6.

## Required separation in outputs
- **Compound-target evidence**
- **Disease / phenotype-target evidence**
- **Overlap / bridge evidence**
- **Network and enrichment evidence**
- **Docking-support evidence**
- **Orthogonal cross-check evidence**
- **Experimental validation evidence** if and only if actually present

## Wording guidance
Use:
- **suggests / prioritizes / is consistent with / nominates / supports a hypothesis**

Avoid:
- **proves / confirms / establishes / demonstrates causal mechanism**

## Common overinterpretation traps
- Treating overlap genes as proven mediators
- Treating hub genes as true master regulators
- Treating enrichment as directional pathway activation proof
- Treating docking as direct biochemical confirmation
- Treating disease-expression support as evidence the exposure caused that pattern
