# Study Patterns

## Pattern A. Exposure–Disease Overlap Workflow
Best when the main deliverable is a plausible molecular bridge between one compound and one disease network.

Core logic:
1. standardize the compound
2. collect compound targets
3. collect disease targets
4. identify overlap genes
5. prioritize hubs
6. enrich pathways
7. synthesize conservatively

## Pattern B. Toxicant-to-Phenotype Mechanism Workflow
Use when the endpoint is an adverse phenotype, organ injury, or toxicity state rather than a tightly named disease.

## Pattern C. Exposure–Disease + Expression Validation Workflow
Best when a public disease dataset can provide orthogonal support for nominated hub genes.

## Pattern D. Endocrine / Metabolic Disruption Workflow
Use only when the biology and literature justify a hormone, steroid, metabolic, or receptor-centered interpretation layer.

## Pattern E. Translational Health-Risk Workflow
Use when the user wants a stricter evidence-downgrade table, reviewer-facing caveats, and a more conservative translational framing.
