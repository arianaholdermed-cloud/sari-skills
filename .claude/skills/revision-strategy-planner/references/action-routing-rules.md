# Action Routing Rules

## Core Rule
Each comment should be routed to the lightest action that credibly solves the problem.

## Possible Action Routes
- new experiment
- new analysis
- re-analysis
- clarification
- wording change
- figure/table revision
- manuscript restructuring
- bounded rebuttal

## Important Rule
Do not jump to new experiments when the real issue is framing, missing explanation, or underuse of existing data.

## Boundary Rule
Do not downgrade a true evidence gap into wording repair just because new data collection is inconvenient.

## Reporting Rule
For each major comment cluster, make the action route explicit.
