# Objective Type Taxonomy

Classify the dominant objective type before rewriting the objective.

## 1. Descriptive Objective
Use when the main purpose is to characterize patterns, distributions, states, or features without testing a directional relationship.

## 2. Association Objective
Use when the goal is to evaluate whether a factor, biomarker, exposure, or feature is related to an outcome or phenotype.

## 3. Prognostic Objective
Use when the objective asks whether a baseline factor predicts a future outcome.

## 4. Predictive / Stratification Objective
Use when the objective aims to classify patients, assign subgroup membership, or predict treatment response or risk.

## 5. Validation Objective
Use when the central purpose is to test whether a previously proposed marker, score, or finding holds in a new setting or cohort.

## 6. Mechanistic Objective
Use when the main question concerns pathway activity, biological process, cell state, or causal biological explanation.

## 7. Comparative Objective
Use when the objective centers on comparison between interventions, assays, groups, models, or approaches.

## 8. Translational Objective
Use when a biological observation is being framed toward diagnosis, prognosis, treatment response, or target relevance.

## Rule
If multiple types are present, identify:
- dominant objective type
- secondary objective type(s)
- what becomes clearer if the objective is rewritten around the dominant type
