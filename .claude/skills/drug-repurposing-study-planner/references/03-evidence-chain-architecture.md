# Evidence Chain Architecture Rules

## Core Rule
Drug repurposing studies must be framed as evidence chains, not disconnected modules.

## Typical Layers
A coherent chain often includes:
1. disease-side biological framing,
2. drug-side relevance,
3. prioritization evidence,
4. orthogonal support,
5. experimental confirmation,
6. translational strengthening.

## Layering Rule
For each layer, specify whether it is:
- necessary,
- recommended,
- or optional.

## Important Rule
Do not promote a downstream claim if the upstream evidence layer is weak or missing.
