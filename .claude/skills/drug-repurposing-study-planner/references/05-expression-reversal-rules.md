# Expression Reversal Rules

## Core Rule
Expression reversal is a hypothesis-generation layer, not proof of efficacy.

## Required Behaviors
When using expression reversal logic:
- define the disease signature source,
- define the perturbation signature source,
- discuss directionality assumptions,
- discuss context mismatch risk,
- and state what orthogonal support is needed.

## Important Rule
Do not treat disease-vs-control reversal as equivalent to disease-modifying therapeutic benefit.
