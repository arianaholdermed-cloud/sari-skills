# Methods Section Writing Guide

## Overview

The Methods section is the core part of a research paper that ensures the reproducibility of the study. This guide focuses on writing a detailed and precise Methods section to ensure research reproducibility and transparency.

## Role of the Methods Section

The Methods section reflects the core of the scientific method:
- **Research Design**: How did you design the study?
- **Participants/Samples**: Who or what did you study?
- **Materials and Equipment**: What did you use?
- **Procedures**: What did you do?
- **Measurement and Analysis**: How did you measure and analyze the data?

---

## The Methods Section

### Purpose

To provide sufficient detail for others to replicate the study and assess its validity.

### Key Principle

Another expert in the field should be able to repeat your experiment exactly as you performed it.

### Standard Subsections

#### Research Design
- State the overall design (e.g., randomized controlled trial, cohort study, cross-sectional survey)
- Justify the design choice if it is not obvious
- Mention blinding, randomization, or controls as applicable

#### Participants/Subjects/Samples
- Define the population of interest
- Precisely describe inclusion and exclusion criteria
- Report the sample size and how it was determined (power analysis)
- Explain recruitment methods and settings
- For animals: specify species, strain, age, sex, and housing conditions

#### Materials and Equipment
- List all materials, reagents, and equipment used
- Include manufacturer names and locations (in parentheses)
- Specify catalog numbers for special items
- Report software names and versions

#### Procedures
- Describe what was done in chronological order
- Include enough detail for replication
- Use subheadings to organize complex procedures
- Specify timing (e.g., "incubated for 2 hours at 37°C")
- For surveys/interviews: describe the instrument, validation, and administration

#### Measurements and Outcomes
- Define all measured variables
- Specify primary and secondary outcomes
- Describe measurement tools and their validity
- Include units of measurement

#### Statistical Analysis
- Name all statistical tests used
- Justify the choice of tests
- State the significance level (usually α = 0.05)
- Report the power analysis for sample size
- Name the statistical software with its version
- Describe the handling of missing data
- Mention adjustments for multiple comparisons if applicable

#### Ethical Considerations
- State IRB/Ethics Committee approval (with approval number)
- Mention informed consent procedures
- For human studies: state adherence to the Declaration of Helsinki
- For animal studies: state adherence to relevant guidelines (e.g., ARRIVE)

### Length
- Typically 2-4 pages
- Proportional to the complexity of the study

### Verb Tense
- Use **past tense** for actions you performed ("We measured...", "Participants completed...")
- Use **present tense** for established procedures ("PCR amplifies...", "The questionnaire consists of...")

### Writing Principles
- Always write in full paragraphs and flowing prose
- Never submit bulleted lists in the final manuscript

### Common Errors
- Lack of detail for replication
- Methods appearing for the first time in the Results section
- Including results or discussion
- Missing statistical tests
- Undefined abbreviations
- Missing ethical approval statements

## Detailed Guide to the Methods Section

### Research Design

#### Basic Elements

**Choice of Study Type:**
- Experimental studies: Randomized Controlled Trials (RCT), quasi-experiments
- Observational studies: Cohort studies, case-control studies, cross-sectional studies
- Descriptive studies: Case reports, case series
- Methodological studies: Diagnostic accuracy, predictive models

**Design Description Requirements:**
- Clearly state the type of research design
- Provide the rationale for the design choice (if not obvious)
- Describe the type of control (if any)
- Explain how blinding was implemented (if any)

#### Randomized Controlled Trials (RCT)

**Essential Elements:**
- Randomization method: Simple randomization, block randomization, stratified randomization
- Allocation concealment: Opaque envelopes, central randomization system
- Blinding: Single-blind, double-blind, triple-blind
- Controls: Placebo control, active control, no-treatment control

**Example Description:**
```
Participants were randomly assigned to the intervention or control group in a 1:1 ratio using a computer-generated random number sequence. 
The allocation sequence was generated by an independent statistician and concealed using opaque, sealed envelopes. 
Participants and researchers remained blinded to the group assignments.
```

#### Observational Studies

**Cohort Studies:**
- Prospective vs. Retrospective
- Inclusion and exclusion criteria
- Follow-up time points and duration
- Handling of loss to follow-up

**Case-Control Studies:**
- Case definition and control selection
- Matching criteria (age, sex, etc.)
- Data collection methods

**Cross-Sectional Studies:**
- Sampling method
- Sample size calculation
- Timeframe

### Participants/Subjects

#### Inclusion Criteria

**Population Definition:**
- Age range
- Sex
- Diagnostic criteria (using recognized standards like ICD-10, DSM-5)
- Disease severity
- Language requirements

**Inclusion Example:**
```
Inclusion criteria were: (1) adults aged 18–65 years; (2) a diagnosis of major depressive disorder according to DSM-5; 
(3) a Hamilton Depression Rating Scale (HAM-D) score ≥ 20; (4) ability to provide informed consent.
```

#### Exclusion Criteria

**Common Reasons for Exclusion:**
- Comorbidities
- Prior treatment history
- Pregnancy or lactation
- Participation in other clinical trials
- Issues with informed consent

**Exclusion Example:**
```
Exclusion criteria included: (1) a history of bipolar or psychotic disorders; (2) current substance use disorder; 
(3) pregnancy or lactation; (4) receipt of electroconvulsive therapy within the past 8 weeks.
```

#### Recruitment Methods

**Recruitment Description:**
- Recruitment sources: Clinics, community advertisements, online registries
- Recruitment timeframe
- Pre-screening methods
- Informed consent acquisition process

**Example:**
```
Participants were recruited from three memory clinics in Boston between January 2022 and December 2023. 
Potential participants were pre-screened via telephone to confirm preliminary eligibility. 
Eligible participants were invited for an informed consent visit, during which study procedures were explained in detail. 
All participants provided written informed consent prior to participation.
```

#### Sample Size Calculation

**Power Analysis Elements:**
- Primary outcome measure
- Expected effect size (based on literature or pilot studies)
- Significance level (α, usually 0.05)
- Power (1-β, usually 0.80 or 0.90)
- Type of test (one-tailed vs. two-tailed)

**Example Description:**
```
Based on previous studies, an expected reduction of 4.5 points (SD=8.0) in the HAM-D score was anticipated for the intervention group. 
Sample size calculation was performed using G*Power 3.1 for a two-independent-sample t-test with α=0.05 and power=0.90. 
The calculated required sample size was 84 per group. 
Accounting for a 20% attrition rate, we planned to recruit 100 participants per group (200 in total).
```

### Materials and Reagents

#### Materials List

**Required Information:**
- Complete list of all materials, reagents, and equipment used
- Manufacturer name
- Manufacturer location (city, country)
- Catalog number (if applicable)
- Lot number (for chemical reagents)
- Software name and version

**Reagent Example:**
```
Cells were cultured in Dulbecco’s Modified Eagle Medium (DMEM; Gibco, Thermo Fisher Scientific, Waltham, MA) 
supplemented with 10% fetal bovine serum (FBS; Gibco) and 1% penicillin-streptomycin (Gibco). 
Antibodies included: anti-β-actin (Cell Signaling Technology, Danvers, MA, #4970, 1:1000 dilution) and 
anti-phospho-Akt (Cell Signaling Technology, #4060, 1:1000 dilution).
```

#### Equipment Description

**Example:**
```
Blood samples were analyzed using an automated clinical analyzer (Cobas 6000, Roche Diagnostics, Basel, Switzerland). 
Centrifugation was performed on a benchtop centrifuge (Sorvall Legend RT+, Thermo Fisher Scientific, Waltham, MA). 
Image acquisition was performed using a confocal microscope (Zeiss LSM 880, Zeiss, Oberkochen, Germany).
```

### Procedures

#### Detailed Step Requirements

**Principles for Describing Procedures:**
- Sufficiently detailed for others to replicate
- Organized chronologically
- Use subheadings for complex procedures
- Include key parameters: temperature, time, concentration, pH
- Indicate the sequence and timing of operations

#### Sample Preparation

**Example (Cell Culture):**
```
HeLa cells were cultured in a humidified incubator at 37°C with 5% CO2. 
Cells were passaged using trypsin-EDTA (Gibco) when they reached 80–90% confluence. 
For experiments, cells were seeded in 6-well plates at a density of 1 × 10^5 cells per well. 
After 24 hours, cells were treated with the specified concentrations of drugs according to the experimental design. 
After 48 hours of treatment, cells were collected for downstream analysis.
```

#### Reagent Preparation

**Example (Buffer):**
```
RIPA lysis buffer was prepared as follows: 50 mM Tris-HCl (pH 7.4), 150 mM NaCl, 
1% NP-40, 0.5% sodium deoxycholate, and 0.1% SDS. 
Protease inhibitor cocktail (Roche, Basel, Switzerland) was added immediately before use. 
The buffer was stored at 4°C and used within 1 week.
```

#### Experimental Procedures

**Example (Western blot):**
```
Cells were lysed using RIPA buffer for 30 minutes. 
Lysates were centrifuged at 14,000 × g for 15 minutes at 4°C. 
Protein concentration was determined using a BCA protein assay kit (Pierce, Rockford, IL). 
30 μg of protein was separated on 4–12% SDS-PAGE gels and transferred to PVDF membranes. 
Membranes were blocked with 5% non-fat milk for 1 hour at room temperature. 
Primary antibodies were incubated overnight at 4°C. 
After washing, HRP-conjugated secondary antibodies were incubated for 1 hour at room temperature. 
Signals were detected using enhanced chemiluminescence (ECL) substrate and the ChemiDoc system.
```

### Measurements and Outcomes

#### Outcome Definitions

**Primary Outcome:**
- The main measurement pre-specified based on the study objective
- Clearly defined
- Valid and reliable measurement

**Secondary Outcomes:**
- Exploratory or supplementary measurements
- Should also be pre-specified in the protocol

**Example:**
```
The primary outcome was the change in HAM-D score from baseline to week 12. 
The HAM-D is a 17-item clinician-rated scale for depression severity, with scores ranging from 0 to 52; higher scores indicate more severe symptoms. 
Secondary outcomes included Hamilton Anxiety Rating Scale (HAM-A) scores, Clinical Global Impression-Improvement (CGI-I) scores, and remission rates (HAM-D ≤ 7).
```

#### Measurement Tools

**Validity Description:**
- Tool name
- Scoring method
- Psychometric properties (reliability, validity)
- Cultural adaptation (if applicable)

**Example:**
```
Depressive symptoms were assessed using the Chinese version of the HAM-D. 
This version has established psychometric properties in the Chinese population, with an internal consistency Cronbach's α = 0.89. 
Ratings were completed by two trained and certified independent raters. 
Raters remained blinded to the treatment allocation of the participants.
```

### Statistical Analysis

#### Analysis Methods

**Description Requirements:**
- Name all statistical tests used
- Justify the choice of tests
- State the test assumptions
- Indicate the software and version used

**Example:**
```
All statistical analyses were performed using R 4.2.1 (R Foundation for Statistical Computing, Vienna, Austria). 
Continuous variables are reported as mean ± standard deviation (SD) or median (interquartile range [IQR]), depending on the distribution. 
Categorical variables are reported as counts and percentages. 
Between-group comparisons were performed using independent-sample t-tests (for normal data) or Mann-Whitney U tests (for non-normal data). 
Categorical variables were analyzed using Chi-square tests or Fisher’s exact tests. 
All tests were two-tailed, and the significance level was set at p < 0.05.
```

#### Handling of Missing Data

**Method Description:**
- Missing data mechanism
- Handling method
- Sensitivity analysis

**Example:**
```
The attrition rate was approximately 15%. 
Missing data analysis indicated a Missing Completely at Random (MCAR) mechanism. 
The primary analysis used complete case analysis. 
Additionally, sensitivity analysis was performed using multiple imputation (MICE package, 5 imputed datasets), yielding similar results.
```

#### Adjustments for Multiple Comparisons

**When performing multiple comparisons:**
- State the adjustment method
- Specify the adjusted significance level

**Example:**
```
Bonferroni correction was applied for pairwise comparisons at three time points. 
The adjusted significance level was p < 0.05/3 = 0.017. 
Additionally, the Benjamini-Hochberg procedure was used to control the false discovery rate (FDR).
```

### Ethical Considerations

#### Human Research

**Required Statements:**
- Ethics committee approval
- Approval number
- Adherence to guidelines (Declaration of Helsinki)
- Informed consent
- Data protection and privacy

**Example:**
```
The study protocol was approved by the Institutional Review Board of [Institution Name] (Approval No.: IRB-2022-00123). 
All procedures followed the Declaration of Helsinki and its later amendments. 
Written informed consent was obtained from all participants prior to participation in any study procedures. 
Participant identities were anonymized and stored confidentially.
```

#### Animal Research

**Required Statements:**
- Ethics committee approval
- Adherence to guidelines (ARRIVE 2.0)
- Animal welfare considerations

**Example:**
```
All animal experiments were approved by the Animal Care and Use Committee of [Institution Name] (Approval No.: IACUC-2022-005). 
Animal care and experimental procedures followed the NIH Guide for the Care and Use of Laboratory Animals. 
This study adhered to the Animal Research: Reporting of In Vivo Experiments (ARRIVE) 2.0 guidelines. 
All procedures were performed under isoflurane anesthesia. 
At the end of the study, animals were humanely euthanized by CO2 asphyxiation.
```

## Common Errors in the Methods Section

### Insufficient Detail

**Problem:** Readers cannot replicate the study.

**Incorrect Example:**
```
Blood samples were collected on ice.
```

**Correction:**
```
5 mL of venous blood was collected from the antecubital vein into EDTA-containing tubes using a 21-gauge needle. 
Tubes were immediately placed on ice after collection and centrifuged at 1,500 × g for 10 minutes within 30 minutes to separate plasma. 
Plasma was aliquoted and stored at -80°C until analysis.
```

### Missing Randomization and Blinding Details

**Incorrect Example:**
```
Participants were randomly assigned to two groups.
```

**Correction:**
```
Participants were randomly assigned to the intervention or control group in a 1:1 ratio using a computer-generated random number sequence (random function in R 4.2.1). 
Randomization was performed by a study coordinator not involved in recruitment or assessment. 
The allocation sequence was concealed using opaque, sealed envelopes. 
Participants and outcome assessors remained blinded to the group assignments.
```

### Missing Sample Size Calculation

**Incorrect Example:**
```
We recruited 100 participants.
```

**Correction:**
```
Based on the effect size reported in previous studies (Cohen's d = 0.6), sample size calculation determined that 64 participants per group were required to detect between-group differences with α = 0.05 and power = 0.80 (two-tailed t-test). 
Accounting for an expected 20% attrition rate, we planned to recruit 80 participants per group (160 in total).
```

### Presenting Results in the Methods Section

**Incorrect Example:**
```
The intervention significantly reduced depressive symptoms (p = 0.002).
```

**Correction:** (Move to the Results section; only describe how to measure and analyze statistically in the Methods)

```
Depressive symptoms were measured using the Hamilton Depression Rating Scale (HAM-D). 
Statistical analysis used independent-sample t-tests to compare the change in HAM-D scores between groups.
```

## Summary of Tense Usage

| Methods Subsection | Verb Tense | Example |
| :--- | :--- | :--- |
| Actions you performed | Past tense | "We measured blood pressure", "Participants completed the questionnaire" |
| Established procedures | Present tense | "PCR amplifies DNA", "The scale consists of 17 items" |

## Adapting Methods for Different Research Types

### Clinical Trials

**Additional Requirements for Methods:**
- Trial registration number and platform
- Detailed description of blinding
- Adverse event reporting methods
- Data and Safety Monitoring Board (if applicable)

### Systematic Reviews and Meta-Analyses

**Method Elements:**
- Search strategy
- Inclusion and exclusion criteria
- Data extraction methods
- Risk of bias assessment methods
- Statistical synthesis methods

### Case Reports

**Methods section is usually called "Case Description":**
- Patient demographics
- Medical history
- Clinical findings
- Diagnostic process
- Interventions

### Observational Studies

**Methods section focuses specifically on:**
- Identification of potential confounding factors
- Confounding control methods (stratification, multivariate adjustment)
- Limitations of causal inference

## Complete Methods Section Example

```
Methods

Research Design

This was a single-center, parallel-group, randomized, placebo-controlled trial. 
The study was conducted at [Hospital Name] from January 2022 to December 2023.

Participants

Inclusion criteria were adults aged 18–65 years with a diagnosis of major depressive disorder according to DSM-5 and a baseline HAM-D score ≥ 20. 
Exclusion criteria included bipolar disorder, psychotic disorders, substance use disorders, pregnancy, or antidepressant treatment within the past 8 weeks.

Participants were recruited from the psychiatric outpatient clinic. 
After eligibility was confirmed, participants provided written informed consent.

Sample size was calculated using G*Power 3.1 based on an effect size d = 0.6, α = 0.05, and power = 0.90. 
The calculation required 64 participants per group. 
Accounting for a 20% attrition rate, 80 participants were recruited per group.

Randomization and Blinding

A computer-generated random number sequence (using R 4.2.1) was used to assign participants to the intervention or placebo group in a 1:1 ratio. 
A study coordinator not involved in recruitment placed the assignments into opaque, sealed envelopes. 
After informed consent, the study coordinator opened the envelope and assigned the participant to the designated group. 
Participants and clinical assessors remained blinded to the group assignments.

Intervention

The intervention group received 50 mg of Compound X (Manufacturer Name, City, Country) once daily for 12 weeks. 
The placebo group received an identical-looking placebo on the same schedule. 
Face-to-face assessments were conducted at baseline, week 4, week 8, and week 12.

Outcomes

The primary outcome was the change in HAM-D score from baseline to week 12. 
The HAM-D is a 17-item clinician-rated scale for depression severity, with scores ranging from 0 to 52. 
Secondary outcomes included HAM-A scores, remission rates (HAM-D ≤ 7), and CGI-I scores.

Statistical Analysis

All analyses were performed using R 4.2.1. 
Continuous variables are reported as mean ± standard deviation (SD) or median [IQR], and categorical variables are reported as counts (percentages). 
The primary analysis used an intention-to-treat (ITT) approach. 
Linear mixed models were used to analyze changes in HAM-D scores over time, with fixed effects for time, treatment, and time × treatment interaction, and random effects for participants. 
Sensitivity analysis was performed using the per-protocol (PP) population. 
All tests were two-tailed, and the significance level was set at p < 0.05.

Ethical Considerations

The study was approved by the Institutional Review Board of [Institution Name] (Approval No.: IRB-2022-00123). 
The study followed the Declaration of Helsinki. 
All participants provided written informed consent prior to participation.
```

## Methods Section Checklist

Before submission, check if the Methods section contains:

### Basic Information
- [ ] Research design clearly described
- [ ] Study timeframe and location
- [ ] Ethics committee approval and approval number
- [ ] Informed consent procedures

### Participants/Samples
- [ ] Inclusion criteria clearly defined
- [ ] Exclusion criteria clearly defined
- [ ] Recruitment methods described
- [ ] Sample size calculation and rationale
- [ ] For animal studies: species, strain, age, sex, and housing conditions

### Materials and Equipment
- [ ] All materials, reagents, and equipment listed
- [ ] Manufacturer names and locations
- [ ] Catalog numbers and lot numbers
- [ ] Software names and versions

### Procedures
- [ ] Detailed description in chronological order
- [ ] Key parameters (temperature, time, concentration, pH) indicated
- [ ] Subheadings used to organize complex procedures
- [ ] Sufficiently detailed for replication

### Measurements and Outcomes
- [ ] Primary outcome clearly defined
- [ ] Secondary outcomes listed
- [ ] Measurement tools described and validated
- [ ] Units of measurement indicated

### Statistical Analysis
- [ ] All statistical tests named
- [ ] Rationale for test selection provided
- [ ] Significance level stated
- [ ] Sample size calculation described (if applicable)
- [ ] Software and version specified
- [ ] Missing data handling methods
- [ ] Adjustments for multiple comparisons (if applicable)

### Ethics and Safety
- [ ] Ethics committee approval number
- [ ] Statement of adherence to guidelines (Helsinki, ARRIVE, etc.)
- [ ] Informed consent procedures
- [ ] Safety precautions (if applicable)
- [ ] Data protection and privacy
