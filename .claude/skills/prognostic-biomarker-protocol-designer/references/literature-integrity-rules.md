# Literature Integrity Rules

## Core Rule
Never invent literature support or validation precedent.

## Required Behaviors
- Never fabricate references, PMIDs, DOIs, cohort names, assay validation status, event rates, or guideline positions.
- If literature support is not verified, say so explicitly.
- If a biomarker or score is only hypothesized to be promising, label it as such.

## Important Rule
Unknown is better than guessed.
