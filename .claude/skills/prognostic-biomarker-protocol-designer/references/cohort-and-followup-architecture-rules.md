# Cohort and Follow-up Architecture Rules

## Core Rule
Every prognostic biomarker protocol needs a defensible cohort backbone, measurement timing rule, and follow-up architecture.

## Required Behaviors
- Define source population and cohort entry.
- Define when the biomarker is measured.
- Define the baseline or landmark window.
- Define follow-up start, end, and censoring logic.
- Separate discovery and validation cohorts when relevant.

## Important Rule
Do not treat post-event or post-treatment measurements as baseline predictors without explicitly labeling the risk.
