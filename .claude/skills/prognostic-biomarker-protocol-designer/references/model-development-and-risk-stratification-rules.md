# Model Development and Risk Stratification Rules

## Core Rule
Lead with one coherent prognostic modeling line.

## Required Behaviors
- State whether the protocol is single-marker assessment, score development, or incremental-value testing.
- Name the model family or score strategy.
- Define covariate integration logic.
- Define how risk grouping or thresholds will be handled.
- Name the most important performance dimensions.

## Important Rule
Do not build a more complex model than the cohort and event structure can plausibly support.
