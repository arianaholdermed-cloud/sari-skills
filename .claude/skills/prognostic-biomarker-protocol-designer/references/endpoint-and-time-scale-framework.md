# Endpoint and Time-Scale Framework

## Core Rule
A prognostic biomarker is only interpretable if the primary endpoint and time scale are operationalized.

## Required Behaviors
- Name one true primary endpoint.
- Define time origin.
- State follow-up horizon.
- Specify whether analysis is time-to-event, fixed-horizon, or longitudinal.
- State competing-risk issues when relevant.

## Important Rule
Do not use vague labels such as “poor prognosis” without an operational endpoint definition.
