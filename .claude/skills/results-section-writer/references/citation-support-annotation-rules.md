# Citation Support Annotation Rules

## Core Rule
When a sentence or context-setting statement clearly needs literature support, add the required citation-support marker and provide a corresponding PubMed search query.

## Common Citation-Needing Statements
- statements about standard practice,
- statements about known disease context,
- statements about prior methodological limitations,
- statements explaining why a validation layer matters,
- statements placing a result block into established context.

## Important Rule
Do not fabricate actual references, PMIDs, or DOIs.
Provide only the marker and a PubMed search query.

## Opt-Out Rule
If the user explicitly says they do not want citation-support annotation, omit it.
