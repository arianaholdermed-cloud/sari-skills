# Full Results Writing Rules

## Core Rule
A full Results section should convert an already clear result hierarchy into disciplined scientific prose.

## Write For
- visible primary findings,
- coherent subsection progression,
- controlled support-analysis placement,
- clean transition between result blocks,
- restrained Results-only language.

## Avoid
- figure-by-figure dumping,
- buried primary findings,
- oversized support sections,
- premature validation emphasis,
- Discussion-style interpretation.

## Important Rule
Do not attempt full-section writing before the underlying result order is sufficiently clear.
